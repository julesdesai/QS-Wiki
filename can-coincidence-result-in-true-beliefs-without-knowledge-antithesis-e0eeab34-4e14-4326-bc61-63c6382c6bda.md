# Can coincidence result in true beliefs without knowledge?

**Node Type:** antithesis
**Node ID:** e0eeab34-4e14-4326-bc61-63c6382c6bda
**Depth:** 4
**Terminal:** No

**Parent:** [No False Lemmas View](no-false-lemmas-view-synthesis-031e6f6f-2b49-47e4-821c-d2421562acef.md)

## Content

**Consider a scenario in which a gambler believes that a specific number will win without any rationale and it wins purely by chance. In such a scenario, intuitively we would judge that even though the belief is true, it does not constitute knowledge because it was based on coincidence.**, **There is a plausible principle that knowledge cannot be based solely on coincidence. This principle together with the event that the gambler's belief is correct by chance entails that true beliefs formed by coincidence do not count as knowledge.**
