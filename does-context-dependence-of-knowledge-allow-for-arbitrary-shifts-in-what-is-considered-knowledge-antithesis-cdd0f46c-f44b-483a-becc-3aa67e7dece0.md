# Does context-dependence of knowledge allow for arbitrary shifts in what is considered knowledge?

**Node Type:** antithesis
**Node ID:** cdd0f46c-f44b-483a-becc-3aa67e7dece0
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist View](contextualist-view-synthesis-55297f9c-50a2-4f24-b1b0-6029f7bef7b2.md)

## Content

**There is a plausible principle that knowledge should remain consistent across similar situations. This principle together with context-dependent standards entails arbitrary variations in what is considered justified, potentially leading to inconsistencies in knowledge claims.**, **A case in which a trivial change in context (e.g., asking a clarification question) shifts the justification standards intuitively challenges the notion that knowledge can be stable in any reliable way.**
