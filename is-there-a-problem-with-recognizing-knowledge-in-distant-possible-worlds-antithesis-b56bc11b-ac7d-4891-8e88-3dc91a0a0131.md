# Is there a problem with recognizing knowledge in distant possible worlds?

**Node Type:** antithesis
**Node ID:** b56bc11b-ac7d-4891-8e88-3dc91a0a0131
**Depth:** 4
**Terminal:** No

**Parent:** [Safety Theory of Knowledge](safety-theory-of-knowledge-synthesis-92f8baec-50d4-4810-a26f-7675d59c578a.md)

## Content

**Consider a scenario where, in all nearby possible worlds, a person holds a belief that remains true, except in highly distant worlds where it turns false under bizarre conditions. Intuitively, such distance should not impact knowledge in the actual world.**, **The principle that knowledge excludes all easily possible falsehoods seems too demanding if we need to consider very distant possible worlds.**
