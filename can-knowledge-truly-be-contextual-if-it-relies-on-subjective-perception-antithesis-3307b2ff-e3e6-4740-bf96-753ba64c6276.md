# Can knowledge truly be contextual if it relies on subjective perception?

**Node Type:** antithesis
**Node ID:** 3307b2ff-e3e6-4740-bf96-753ba64c6276
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism](contextualism-synthesis-b8a53bd2-d0b8-44ba-a658-8cb1f890a45a.md)

## Content

**Consider a scenario in which two people observe the same event but interpret it differently due to differing contexts. In such a scenario, intuitively, we would not judge that both individuals possess the same knowledge of the event.**, **A plausible principle is that knowledge should be consistent across different contexts if it is accurate. This principle, together with the view that context affects perception, entails a potential inconsistency in knowledge claims.**, **It is implausible that knowledge can vary significantly between observers yet still be considered objective knowledge.**
