# Can one still have knowledge if cognitive faculties fail in a conducive environment?

**Node Type:** antithesis
**Node ID:** 899212ec-8f73-4298-a9f7-40d47256cb03
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-66a6c079-b861-44cf-ac2d-ed1fae76b934.md)

## Content

**Consider a scenario in which a person's cognitive faculties malfunction, yet the person forms a true belief in an environment usually conducive to truth. In such a scenario, intuitively we would judge that the person does not have knowledge.**, **There is a plausible principle that knowledge requires reliable cognitive faculties. This principle together with the idea that cognitive malfunction is possible even in conducive environments entails that proper function may not be necessary for knowledge.**
