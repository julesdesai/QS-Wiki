# Is knowledge context-dependent in all cases?

**Node Type:** antithesis
**Node ID:** d46c4184-4189-4cd7-b9fe-a44e4ca9eed2
**Depth:** 4
**Terminal:** No

**Parent:** [Standards-Based Contextualism](standards-based-contextualism-synthesis-00ac39b8-e543-4e05-a594-bb8273954856.md)

## Content

**It is implausible that knowledge can be entirely context-dependent, since certain objective facts remain true across all contexts**, **If knowledge is context-dependent, then the same proposition can be true and not true simultaneously, which is logically inconsistent**
