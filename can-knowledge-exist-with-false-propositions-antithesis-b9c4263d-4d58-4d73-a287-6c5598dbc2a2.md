# Can knowledge exist with false propositions?

**Node Type:** antithesis
**Node ID:** b9c4263d-4d58-4d73-a287-6c5598dbc2a2
**Depth:** 4
**Terminal:** No

**Parent:** [Defeasibility Theory](defeasibility-theory-synthesis-bec695a8-d154-4145-af72-ab6869088e07.md)

## Content

**Consider a case in which a person is an expert meteorologist who predicts rain tomorrow based on available data, and it is indeed true when it rains the next day. Intuitively, we accept that they know it will rain. However, suppose data changes which if known would have shown it may not rain, suggesting the proposition was not true beforehand.**, **There is a plausible principle that expert judgment based on best available data constitutes knowledge even if underlying evidence later appears flawed. This principle suggests that true belief is not always necessary prior to final verification.**
