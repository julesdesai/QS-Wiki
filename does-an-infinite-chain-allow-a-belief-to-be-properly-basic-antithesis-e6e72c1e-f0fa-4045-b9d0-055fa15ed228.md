# Does an infinite chain allow a belief to be properly basic?

**Node Type:** antithesis
**Node ID:** e6e72c1e-f0fa-4045-b9d0-055fa15ed228
**Depth:** 4
**Terminal:** No

**Parent:** [Infinitism and Non-Foundational Justification](infinitism-and-non-foundational-justification-synthesis-6eeef37c-3f39-4c5c-91c9-b90fbd483157.md)

## Content

**A belief that is part of an infinite chain cannot be foundational, yet intuitively, some beliefs seem self-evident or properly basic without needing further justification.**, **A principle of properly basic beliefs suggests some beliefs can be justified without reference to an infinite chain, and the irrelevance of infinite chains for these circumscribes the need for such chains.**
