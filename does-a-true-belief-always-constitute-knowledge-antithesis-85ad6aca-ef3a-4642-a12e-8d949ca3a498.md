# Does a true belief always constitute knowledge?

**Node Type:** antithesis
**Node ID:** 85ad6aca-ef3a-4642-a12e-8d949ca3a498
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology: Intellectual Virtues](virtue-epistemology-intellectual-virtues-synthesis-1d322156-3f22-49a0-99a5-f932d24b0e66.md)

## Content

**Consider a case in which someone guesses the outcomes of dice rolling correctly multiple times. Though the belief is true, the intuitive judgment is that this is not knowledge.**, **There is a plausible principle that knowledge requires more than just truth and belief, that the true belief should be non-accidental, thereby challenging the idea that true belief is sufficient for knowledge.**
