# Can true belief based on lucky guesses count as knowledge?

**Node Type:** antithesis
**Node ID:** 3abd017b-a74f-4222-ae06-d0b058b7f0e5
**Depth:** 4
**Terminal:** No

**Parent:** [Externalist Foundationalism](externalist-foundationalism-synthesis-3e0224db-6723-4288-8a1b-8819cbac7570.md)

## Content

**Suppose we have a case in which a person guesses a true proposition correctly about a future event without any basis other than a lucky guess. Intuitively, we either have to say that guessing correctly constitutes knowledge or that it does not. If we say guessing correctly constitutes knowledge, then it trivializes the notion of knowledge. If we say it does not, then true belief alone is insufficient for knowledge.**
