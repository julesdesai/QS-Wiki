# Does coherentism allow for justified false beliefs?

**Node Type:** antithesis
**Node ID:** af99655a-7111-42ef-a7f9-ebc3c2a437a6
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism Summary](coherentism-summary-synthesis-d4794edf-a9d9-4687-b207-9409b552ac03.md)

## Content

**A set of beliefs that are internally coherent can still contain false beliefs.**, **If a coherent set of beliefs can include false beliefs, then coherence alone is insufficient for justification.**, **It is plausible that justified beliefs should also be true beliefs.**
