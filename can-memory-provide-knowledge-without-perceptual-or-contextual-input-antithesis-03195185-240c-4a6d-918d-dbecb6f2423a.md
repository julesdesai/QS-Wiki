# Can memory provide knowledge without perceptual or contextual input?

**Node Type:** antithesis
**Node ID:** 03195185-240c-4a6d-918d-dbecb6f2423a
**Depth:** 4
**Terminal:** No

**Parent:** [Contextual Phenomenalism](contextual-phenomenalism-synthesis-7d8a3858-3738-47ab-85b6-1a71b73ecaf5.md)

## Content

**A case in which someone accurately recalls past events without current perceptual experiences challenges the dependency of knowledge on present perception.**, **There is a plausible principle that memory can serve as a source of knowledge without the need for contextual interpretation.**, **It is implausible that all knowledge requires current perceptual or contextual input if past information can be accurately retained and known through memory alone.**
