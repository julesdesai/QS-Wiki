# Can justified beliefs be false?

**Node Type:** antithesis
**Node ID:** 1509d800-ba41-430e-b647-26f75b79d8eb
**Depth:** 4
**Terminal:** No

**Parent:** [Evidentialism Summary](evidentialism-summary-synthesis-dea02623-290c-44c3-896b-2f967081eb1b.md)

## Content

**A belief can be justified if it is backed by supporting evidence, yet that evidence could be misleading or incorrect, leading to a false belief.**, **Suppose there is a case where a person has collected evidence that is seemingly solid, but unbeknownst to them, it is false. Intuitively, we would say the person's belief is justified despite being false.**
