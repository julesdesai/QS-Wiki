# Is it possible for a belief to be unsafe yet still constitute knowledge?

**Node Type:** antithesis
**Node ID:** 67ab46fa-2101-4098-9b73-e9ad4b87697f
**Depth:** 4
**Terminal:** No

**Parent:** [Safety Condition Theory](safety-condition-theory-synthesis-b5a72759-42a3-4d9b-b74c-df40c84bdc9d.md)

## Content

**A scenario in which a person has a true belief that is based on an unreliable method can occur. In such cases, intuitively, we might still judge that the person has knowledge.**, **A plausible principle is that knowledge can be attained through a lucky guess. This principle suggests that some knowledge can exist even if the belief could easily have been false.**
