# Is unconscious inference compatible with justified true belief?

**Node Type:** antithesis
**Node ID:** 156476de-2825-44b0-b78b-16b478935f1e
**Depth:** 4
**Terminal:** No

**Parent:** [Indirect Realism with a Role for Unconscious Inference](indirect-realism-with-a-role-for-unconscious-inference-synthesis-78a86651-c747-4945-9c1d-0df1bc77c499.md)

## Content

**Suppose we have a case where a person unconsciously infers a true belief about the world without being able to justify it. Intuitively, a true belief without justification cannot constitute knowledge.**, **There is a plausible principle asserting that knowledge requires justified true belief, which entails a different requirement from unconscious inference, suggesting a conflict with defining knowledge.**
