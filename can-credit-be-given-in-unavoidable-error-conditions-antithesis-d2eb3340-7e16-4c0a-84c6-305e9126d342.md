# Can credit be given in unavoidable error conditions?

**Node Type:** antithesis
**Node ID:** d2eb3340-7e16-4c0a-84c6-305e9126d342
**Depth:** 4
**Terminal:** No

**Parent:** [Credit Theory of Knowledge](credit-theory-of-knowledge-synthesis-f65ebe79-7fd3-4c3c-9fb3-cd47f9656422.md)

## Content

**It is implausible that an agent can deserve credit if an element of their belief-forming process is systematically prone to error. If an agent can gain knowledge credit while using unreliable methods, the link between intellectual virtue and knowledge becomes tenuous.**
