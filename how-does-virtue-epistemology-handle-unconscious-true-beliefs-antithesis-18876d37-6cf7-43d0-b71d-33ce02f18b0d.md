# How does virtue epistemology handle unconscious true beliefs?

## Content

**Imagine a person has a true belief that they are not consciously aware of, such as accurately estimating a distance unconsciously.**, **An unconscious belief does not actively involve an intellectual virtue as it requires no conscious cognitive effort.**, **Conscious reflection or reasoning is often necessary to exercise cognitive virtue.**
