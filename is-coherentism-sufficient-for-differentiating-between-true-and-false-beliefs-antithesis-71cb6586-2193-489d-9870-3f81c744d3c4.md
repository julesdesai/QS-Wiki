# Is coherentism sufficient for differentiating between true and false beliefs?

**Node Type:** antithesis
**Node ID:** 71cb6586-2193-489d-9870-3f81c744d3c4
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentist Phenomenalism](coherentist-phenomenalism-synthesis-7f881170-8b02-4aad-92a8-2bed5a4b8a3c.md)

## Content

**Consider a belief system based on coherent but entirely fictional narratives, like an internally consistent set of beliefs about a non-existent planet.**, **Coherence alone may allow these fictional beliefs to be justified, though they do not represent knowledge about the actual world.**, **Knowledge that is only supported by coherence may not effectively differentiate between truth and fiction.**
