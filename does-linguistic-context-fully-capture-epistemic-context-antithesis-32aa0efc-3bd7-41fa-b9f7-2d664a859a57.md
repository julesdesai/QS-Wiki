# Does linguistic context fully capture epistemic context?

**Node Type:** antithesis
**Node ID:** 32aa0efc-3bd7-41fa-b9f7-2d664a859a57
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism](contextualism-synthesis-876f0403-5dcc-4837-9485-46808d567bde.md)

## Content

**Consider a scenario in which linguistic context changes but the epistemic facts remain constant. Intuitively, we might judge that knowledge should be rooted in epistemic facts rather than linguistic interpretations.**, **It is implausible that a change in mere linguistic context could affect whether someone knows a proposition.**
