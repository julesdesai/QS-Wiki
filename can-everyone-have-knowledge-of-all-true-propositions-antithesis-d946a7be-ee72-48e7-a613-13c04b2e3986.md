# Can everyone have knowledge of all true propositions?

**Node Type:** antithesis
**Node ID:** d946a7be-ee72-48e7-a613-13c04b2e3986
**Depth:** 4
**Terminal:** No

**Parent:** [Refined Virtue Epistemology](refined-virtue-epistemology-synthesis-ed8e6ab8-d432-4955-9aec-982a4b148be7.md)

## Content

**Suppose we have a case in which a person believes all true propositions but does so without any intellectual virtue or engagement with the content. Intuitively, not all of these beliefs can count as knowledge.**, **Knowledge is true belief arising from intellectual virtue, not merely any true belief. It requires active engagement and intellectual effort.**
