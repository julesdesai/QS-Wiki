# Does skill-based knowledge undermine propositional truth?

**Node Type:** antithesis
**Node ID:** 1dbab800-9039-420f-b7d1-ac9811501f2a
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology with Skills](virtue-epistemology-with-skills-synthesis-458da717-0537-4f5a-8a52-aec5d63f6438.md)

## Content

**Suppose we have a case where a person can perform a skill impeccably but believes incorrect propositions about the underlying practice. Intuitively, we either have to say that skill operation makes incorrect beliefs permissible for knowledge, or that true propositional beliefs are necessary. If skill with false beliefs suffices, it challenges the role of true propositions in knowledge. If true beliefs are necessary, it questions the sufficiency of skills alone.**
