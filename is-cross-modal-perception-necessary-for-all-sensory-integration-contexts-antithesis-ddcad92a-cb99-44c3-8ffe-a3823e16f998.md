# Is cross-modal perception necessary for all sensory integration contexts?

**Node Type:** antithesis
**Node ID:** ddcad92a-cb99-44c3-8ffe-a3823e16f998
**Depth:** 4
**Terminal:** No

**Parent:** [Embodied Cognition with Sensory Integration](embodied-cognition-with-sensory-integration-synthesis-1b01035b-41e7-4892-81fb-893d75aaae6e.md)

## Content

**Suppose we have a case in which an individual experiences integrated sensory inputs within one modality, such as hearing different musical notes as a single chord. Intuitively, we might say that this integration does not involve cross-modal perception.**, **It is implausible that cross-modal perception explains all sensory integration contexts.**
