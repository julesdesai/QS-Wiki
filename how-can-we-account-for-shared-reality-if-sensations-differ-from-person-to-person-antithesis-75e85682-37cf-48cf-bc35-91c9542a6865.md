# How can we account for shared reality if sensations differ from person to person?

**Node Type:** antithesis
**Node ID:** 75e85682-37cf-48cf-bc35-91c9542a6865
**Depth:** 4
**Terminal:** No

**Parent:** [Phenomenalism Summary](phenomenalism-summary-synthesis-11423d6b-b29a-46b2-967d-bb4b03435a75.md)

## Content

**Consider a scenario where person A sees an object as blue, while person B sees it as green in identical conditions. In such a scenario, intuitively we would judge that there is an objective way the object is that is independent of these perceptions.**, **It is implausible that a purely phenomenological approach can account for the shared agreement on the features of external objects when individuals' sensory experiences differ.**
