# Can unverifiable truths still hold meaning?

**Node Type:** antithesis
**Node ID:** 72b938b4-6d2b-496a-bb32-84aa8bf73dcd
**Depth:** 4
**Terminal:** No

**Parent:** [Verificationism](verificationism-synthesis-742a7e84-42a2-4d13-8c35-af2e03d93250.md)

## Content

**A statement about the past, such as "dinosaurs existed," even if true, cannot be empirically verified today through direct observation**, **If the principle that a statement only has meaning if it can be empirically verified is true, then statements about past events we cannot verify today would be meaningless**, **It is implausible that statements about the past, which are widely accepted, are meaningless**
