# Does naturalized epistemology allow for subjective experiences to count as knowledge?

**Node Type:** antithesis
**Node ID:** f98c02ef-b94d-4a40-9720-8829da3f329f
**Depth:** 4
**Terminal:** No

**Parent:** [Naturalized Epistemology Summary](naturalized-epistemology-summary-synthesis-ad83a102-e1e6-4596-90b6-8b3b07d31230.md)

## Content

**Suppose a case where a person has a strong personal experience or insight that leads to a true belief, but it cannot be independently verified by scientific methods. Intuitively, many would say this should count as knowledge**, **It is implausible that knowledge excludes personal insights that cannot be captured by scientific epistemology, as this ignores well-established forms of understanding**
