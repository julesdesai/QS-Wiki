# Is awareness of cognitive task performance necessary for knowledge?

**Node Type:** antithesis
**Node ID:** 5229cd13-770e-4297-bd61-7282762e87c6
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Functionalism with Cognitive Task Fulfillment](epistemic-functionalism-with-cognitive-task-fulfillment-synthesis-3edd3165-e9fd-4e05-81df-6b63dbbb89eb.md)

## Content

**Suppose a person expertly completes a task but is unaware they did so because they were in a distracted state. Intuitively, it seems that knowledge requires some awareness of one's cognitive processes.**, **The idea that awareness is part of knowing suggests that task fulfillment in a distracted state might not constitute real knowledge.**
