# Is there a way to have true belief without properly functioning faculties?

**Node Type:** antithesis
**Node ID:** 38d64bc0-a95e-49de-895b-a050ec12c5e6
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Functionalism](proper-functionalism-synthesis-c17f2c11-845a-40b6-a6a2-0a1552f50f75.md)

## Content

**A case where a person has a true belief because of a lucky guess is not a case where their belief is produced by properly functioning cognitive faculties.**, **A person can have a true belief through sheer coincidence without their cognitive faculties functioning according to any design plan.**
