# Does acquiring true belief always require intellectual virtue for it to be knowledge?

**Node Type:** antithesis
**Node ID:** 11577326-9b4e-43b3-8615-7ed22e1c3088
**Depth:** 4
**Terminal:** No

**Parent:** [Mixed Virtue Epistemology](mixed-virtue-epistemology-synthesis-9b1d3c06-dd01-4be5-8e49-871c85befc1a.md)

## Content

**Suppose we have a case in which someone accidentally guesses a true proposition without employing any discernible intellectual virtue. Intuitively, we would not count this as knowledge even though the belief is true.**, **There is an implication that intellectual virtue involves active cognitive engagement or skill, which is missing in mere guessing. This suggests a potential gap between true belief and knowledge if intellectual virtue is required.**
