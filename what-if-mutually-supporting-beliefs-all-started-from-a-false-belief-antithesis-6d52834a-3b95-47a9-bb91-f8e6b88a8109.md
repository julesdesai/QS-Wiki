# What if mutually supporting beliefs all started from a false belief?

**Node ID:** 6d52834a-3b95-47a9-bb91-f8e6b88a8109

## Content

**If each belief in an infinite network is justified by another, but the foundational belief is false, coherence may only amplify the error rather than providing actual justification.**, **In a scenario where an error is at the base of a coherent belief network, added justification through coherence doesn't remedy the falsehood.**
