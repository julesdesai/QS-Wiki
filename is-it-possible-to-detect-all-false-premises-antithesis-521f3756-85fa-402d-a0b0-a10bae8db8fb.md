# Is it possible to detect all false premises?

## Content

**There is a plausible principle that not all false premises in a belief system are detectable.**, **Suppose we have a case in which a person holds a justified true belief based on premises that cannot be verified. Intuitively, this might suggest that undetected false premises could still underlie the belief.**
