# Can someone be justified in a false belief if the process seems reliable?

**Node Type:** antithesis
**Node ID:** b7393ef4-04d3-473d-98e8-9fab65c4c022
**Depth:** 4
**Terminal:** No

**Parent:** [Externalist Justification View](externalist-justification-view-synthesis-0d99dd10-4e3a-47d1-98be-f70a67b59dde.md)

## Content

**If a process seems reliable but produces a false belief, intuitively, it feels unjustified to claim the belief is also justified.**, **A person uses a seemingly reliable process, forming a false belief; intuitively, we do not consider this knowing.**
