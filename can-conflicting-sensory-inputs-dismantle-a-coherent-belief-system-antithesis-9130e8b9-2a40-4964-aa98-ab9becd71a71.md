# Can conflicting sensory inputs dismantle a coherent belief system?

**Node Type:** antithesis
**Node ID:** 9130e8b9-2a40-4964-aa98-ab9becd71a71
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism in Perception with Integrated Senses](coherentism-in-perception-with-integrated-senses-synthesis-6f261103-b521-4b0f-9f10-d6388a66a95d.md)

## Content

**Suppose one sense—like vision—provides information that an object is red, while another—like touch—indicates it is smooth.**, **If these sensory inputs conflict in color-blind or texture-insensitive situations and still fit into a coherent belief system, it is implausible that these beliefs can be genuinely justified.**
