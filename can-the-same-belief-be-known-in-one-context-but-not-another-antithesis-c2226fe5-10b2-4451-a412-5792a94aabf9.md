# Can the same belief be known in one context but not another?

**Node Type:** antithesis
**Node ID:** c2226fe5-10b2-4451-a412-5792a94aabf9
**Depth:** 4
**Terminal:** No

**Parent:** [Subject-Sensitive Invariantism Summary](subject-sensitive-invariantism-summary-synthesis-519a57d4-27a1-46ac-ac38-c130e30ad088.md)

## Content

**Consider a scenario where a person believes a true proposition in both high-stakes and low-stakes contexts. Intuitively, if knowledge standards are invariant, the proposition should be known or unknown in both contexts equally.**, **A plausible principle is that identical beliefs in different practical contexts should have identical knowledge status if standards do not vary.**
