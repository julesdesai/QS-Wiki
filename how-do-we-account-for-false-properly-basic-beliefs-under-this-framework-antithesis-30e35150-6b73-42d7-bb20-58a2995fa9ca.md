# How do we account for false properly basic beliefs under this framework?

**Node Type:** antithesis
**Node ID:** 30e35150-6b73-42d7-bb20-58a2995fa9ca
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Foundationalism](proper-foundationalism-synthesis-115cfa71-603e-48b1-bce9-39f7dced12e3.md)

## Content

**Suppose we have a case in which a person holds a properly basic belief that is false due to misinformation or misunderstanding. Intuitively, we either have to say that such a belief is justified or that it is not properly basic. If we say it is justified, then false beliefs could be foundational, undermining knowledge. If we say it is not properly basic, then the requirement of proper foundationalism fails to clearly delineate foundational beliefs.**, **There is a plausible principle that foundational beliefs must be both basic and true to justify knowledge. This principle together with the claim of properly basic foundational beliefs entails that all foundational beliefs must also be true.**, **It is implausible to exclude the possibility of basic beliefs being false, thus raising concerns about their role in knowledge claims.**
