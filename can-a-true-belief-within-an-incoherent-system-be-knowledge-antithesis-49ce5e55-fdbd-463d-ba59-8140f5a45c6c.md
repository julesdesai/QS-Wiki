# Can a true belief within an incoherent system be knowledge?

**Node Type:** antithesis
**Node ID:** 49ce5e55-fdbd-463d-ba59-8140f5a45c6c
**Depth:** 4
**Terminal:** No

**Parent:** [Reliable Coherentism](reliable-coherentism-synthesis-cd562728-46bb-4ccf-9e19-2bbd570f1b0e.md)

## Content

**A belief may be true within an overall incoherent system, such as holding a correct scientific belief amidst generally unfounded superstitions.**, **Truth does not imply coherence; thus, having a correct belief in an incoherent system suggests truth is not sufficient for knowledge.**
