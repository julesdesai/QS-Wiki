# Is cognitive processing necessary for all knowledge?

**Node Type:** antithesis
**Node ID:** e5893e23-ebe0-4dcb-a050-b6a1ae8eaae6
**Depth:** 4
**Terminal:** No

**Parent:** [Indirect Realism Summary](indirect-realism-summary-synthesis-8a578877-1ef2-4cd7-a175-3a668ae7b8a0.md)

## Content

**Suppose we have a case in which a child learns a simple fact directly without any apparent cognitive processing, such as “fire is hot” after touching a flame. Intuitively, we would say the child knows this fact.**, **It is implausible that knowledge always requires perception after cognitive processing, as in some instances this processing might be minimal or absent yet knowledge still exists.**
