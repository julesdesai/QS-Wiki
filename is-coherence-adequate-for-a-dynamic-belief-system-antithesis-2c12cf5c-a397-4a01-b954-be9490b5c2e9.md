# Is coherence adequate for a dynamic belief system?

**Node ID:** 2c12cf5c-a397-4a01-b954-be9490b5c2e9

**Parent:** [Minimal Foundational Coherentism Summary](minimal-foundational-coherentism-summary-synthesis-0e7aa745-4942-4fbb-a96c-6043bbe35e5d.md)

## Content

**Consider a belief system where new evidence must be integrated to maintain coherence. Intuitively, certain coherent systems might resist integration of new contradictory evidence.**, **Resistance to incorporating new evidence implies that coherence is not sufficient for a dynamically justified system.**
