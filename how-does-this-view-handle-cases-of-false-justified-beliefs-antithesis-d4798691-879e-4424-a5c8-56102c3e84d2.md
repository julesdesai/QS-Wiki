# How does this view handle cases of false justified beliefs?

**Node Type:** antithesis
**Node ID:** d4798691-879e-4424-a5c8-56102c3e84d2
**Depth:** 4
**Terminal:** No

**Parent:** [Hybrid Internalism](hybrid-internalism-synthesis-3d9ce0d9-6ba2-4665-a054-ed06708d6ee0.md)

## Content

**Consider a case where a person has a fully justified belief that turns out to be false due to deceptive information. Intuitively, we would judge this belief does not count as knowledge. This suggests that there must be more to knowledge than accessible justification and truth alone.**
