# Can safe beliefs be false in nearby worlds?

**Node Type:** antithesis
**Node ID:** eec61741-4ac8-4f73-9029-e1a416017ef7
**Depth:** 4
**Terminal:** No

**Parent:** [Safety Theory](safety-theory-synthesis-3968e05f-d744-409d-bfbb-59e5b6ab2a0b.md)

## Content

**A belief is considered "safe" if it is true in most nearby possible worlds where it is held.**, **It is possible for a belief to be false in the actual world while true in most nearby possible worlds.**, **If a belief can be false in the actual world but true in most nearby worlds, then it is not necessarily true.**
