# Can we have competing coherent systems?

**Node Type:** antithesis
**Node ID:** 9c583e3b-60c2-41ec-a6f9-797dd59ca3b3
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism Summary](coherentism-summary-synthesis-189c934d-40a0-41d9-b70b-1a7a0a9cc7c3.md)

## Content

**Suppose we have a case where competing scientific theories are internally coherent yet provide conflicting explanations of the same phenomena. Intuitively, we either have to say both theories are true or only one is true. If we say both are true, we accept contradictions. If only one is true, coherence does not determine which.**
