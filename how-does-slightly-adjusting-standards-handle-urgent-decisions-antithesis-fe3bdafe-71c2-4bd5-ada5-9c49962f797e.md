# How does slightly adjusting standards handle urgent decisions?

## Content

**Suppose an urgent decision has to be made, with quickly changing information, like during a natural disaster. Intuitively, slightly adjusting standards is inadequate for rapid, high-stakes decisions.**, **There is a plausible principle that significant changes in standards are necessary to handle urgency and unpredictability effectively.**
