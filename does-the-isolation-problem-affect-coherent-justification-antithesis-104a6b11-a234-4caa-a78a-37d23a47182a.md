## Content

**Isolated coherent belief systems, with no input from external reality, are implausible candidates for securing knowledge.**, **A coherent belief system that develops independently of empirical input fails to ensure true belief as knowledge.**, **There is a plausible principle that an external, empirical component is essential for belief to be considered knowledge, which mere coherence does not provide.**
