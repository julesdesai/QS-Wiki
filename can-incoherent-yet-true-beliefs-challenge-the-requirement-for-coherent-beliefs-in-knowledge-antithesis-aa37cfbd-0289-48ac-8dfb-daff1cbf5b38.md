# Can incoherent yet true beliefs challenge the requirement for coherent beliefs in knowledge?

**Node Type:** antithesis
**Node ID:** aa37cfbd-0289-48ac-8dfb-daff1cbf5b38
**Depth:** 4
**Terminal:** No

**Parent:** [Partial Correspondence Coherentism Summary](partial-correspondence-coherentism-summary-synthesis-73282c9a-d730-43f4-a933-7c2ffa7c9bab.md)

## Content

**Imagine a person who has beliefs that are each true and correspond to reality but the set is incoherent, such as owing to conflicting information sources.**, **Intuitively, these beliefs still represent actual states of affairs.**, **If factual correspondence suffices for knowledge, coherence with the belief system might be unnecessary.**
