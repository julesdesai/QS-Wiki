# Can a true belief supported by inadequate evidence still count as knowledge?

**Node Type:** antithesis
**Node ID:** 4926bb7f-08bf-4f39-9f62-7c3179d5ea4d
**Depth:** 4
**Terminal:** No

**Parent:** [Evidentialist View](evidentialist-view-synthesis-6f63629b-c949-4605-9acc-d91a2708886a.md)

## Content

**Consider a scenario in which a person has a true belief, but the belief is supported only by inadequate evidence, such as a lucky guess. Intuitively, we might judge that this is still knowledge because the belief is true.**, **There is a plausible principle that a true belief that reliably leads to truth ought to be considered knowledge, regardless of the adequacy of the evidence. This principle, coupled with cases of true belief arising from unreliable methods, suggests that some knowledge might exist outside of evidential criteria.**
