# How do we account for perceptual error?

**Node Type:** antithesis
**Node ID:** f4dcf727-5caf-4fdf-bbf2-4f2ce182d24f
**Depth:** 4
**Terminal:** No

**Parent:** [Coherent Phenomenalism](coherent-phenomenalism-synthesis-2365a190-e3fc-4d37-a83b-defd0eadffa2.md)

## Content

**Suppose a person perceives an object as blue due to lighting conditions that make it appear blue, and this perception fits within their web of experiences coherently. Intuitively, knowledge about the object's true color is not achieved due to the error in color perception.**, **It is implausible that perceptual errors, even when they fit coherently within a set of experiences, can lead to true knowledge about external objects.**
