# Can a system of consistent yet mutually unfounded beliefs count as knowledge?

**Node Type:** antithesis
**Node ID:** 22293251-953b-42c5-95cf-9f95a8cbb09f
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism: System of Beliefs](coherentism-system-of-beliefs-synthesis-446f5d76-49b9-4fbc-a49a-1aaf884652f2.md)

## Content

**A system can be coherent when all beliefs support each other even if none is independently justified.**, **If no belief within the system is independently justified, the coherence does not add to their justification.**, **Knowledge requires justified true belief.**, **Therefore, a system of mutually unfounded beliefs, regardless of coherence, does not amount to knowledge.**
