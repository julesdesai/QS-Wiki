# Can one possess knowledge if task fulfillment is based on unreliable cognitive functions?

**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Functionalism with Cognitive Task Fulfillment](epistemic-functionalism-with-cognitive-task-fulfillment-synthesis-3edd3165-e9fd-4e05-81df-6b63dbbb89eb.md)

## Content

**Imagine a person has a cognitive disorder that randomly lets them perform tasks accurately but without consistency. Intuitively, we might question whether any success here constitutes knowledge.**, **A plausible principle is that reliability over time is necessary for real knowledge, suggesting unreliable task fulfillment is insufficient.**
