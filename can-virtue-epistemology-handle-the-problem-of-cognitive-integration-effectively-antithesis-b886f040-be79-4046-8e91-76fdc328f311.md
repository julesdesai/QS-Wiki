# Can virtue epistemology handle the problem of cognitive integration effectively?

**Node Type:** antithesis
**Node ID:** b886f040-be79-4046-8e91-76fdc328f311
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-1adc9ad4-a545-4f75-9435-0709a79c765f.md)

## Content

**Consider a scenario where a person uses advanced technology that they do not understand to acquire true beliefs, intuitively raising doubts about the presence of knowledge.**, **Intellectual virtue, if it requires understanding of cognitive processes like using technology, cannot account for knowledge in cases involving external aids unintruiged to an agent's cognitive faculties.**, **If acquiring knowledge requires crediting truths to cognitive abilities, this is problematized by reliance on cognitive prosthetics that obscure individual's abilities.**
