# Is infinite cognitive processing necessary for coherentism to work?

**Node Type:** antithesis
**Node ID:** 1a5ebc43-7bcd-4fcd-8011-e40a581b7c3d
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism with Infinite Justificatory Network](coherentism-with-infinite-justificatory-network-synthesis-c0ed2adb-3371-4bf3-b36f-58a215809fa4.md)

## Content

**A belief is only justified if it can be processed cognitively within the network that supports it.**, **If the justificatory network is infinite, achieving cognitive processing for all connections in the network requires infinite cognitive ability, which is implausible for human cognition.**
