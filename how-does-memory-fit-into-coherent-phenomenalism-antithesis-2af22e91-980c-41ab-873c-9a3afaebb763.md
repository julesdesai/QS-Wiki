# How does memory fit into coherent phenomenalism?

**Node Type:** antithesis
**Node ID:** 2af22e91-980c-41ab-873c-9a3afaebb763
**Depth:** 4
**Terminal:** No

**Parent:** [Coherent Phenomenalism](coherent-phenomenalism-synthesis-2365a190-e3fc-4d37-a83b-defd0eadffa2.md)

## Content

**Consider a scenario in which a person's memory fails, but they generate new, coherent perceptual experiences that contradict their earlier memories. Intuitively, such a person cannot claim to know propositions based on both incompatible sets of experiences.**, **There is a plausible principle that reliable memory is necessary for continuous knowledge over time, which is not accounted for by mere coherence at a single point.**
