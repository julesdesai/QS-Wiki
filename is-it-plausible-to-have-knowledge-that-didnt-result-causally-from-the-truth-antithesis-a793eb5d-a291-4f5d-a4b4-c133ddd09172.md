# Is it plausible to have knowledge that didn’t result causally from the truth?

**Node Type:** antithesis
**Node ID:** a793eb5d-a291-4f5d-a4b4-c133ddd09172
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Externalism](epistemic-externalism-synthesis-a9c372ef-1ae7-460b-b26c-8a82dc6a1ee1.md)

## Content

**Suppose we have a case in which a belief aligns with the truth purely coincidentally, without any causal interaction.**, **Intuitively, we either have to say that this is knowledge or that it is not knowledge.**, **If we say it is knowledge, then we deny that understanding or interaction with the truth is essential.**, **If we deny it is knowledge, then a causal connection to the truth seems necessary for knowledge.**
