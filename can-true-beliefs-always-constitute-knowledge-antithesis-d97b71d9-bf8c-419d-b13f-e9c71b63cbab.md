# Can true beliefs always constitute knowledge?

**Node Type:** antithesis
**Node ID:** d97b71d9-bf8c-419d-b13f-e9c71b63cbab
**Depth:** 4
**Terminal:** No

**Parent:** [Invariantism with Fixed Standards](invariantism-with-fixed-standards-synthesis-2877eecc-8388-4e01-b87d-ffe7f20b2619.md)

## Content

**Two people can both believe the same true proposition with the same evidence, yet intuitively only one might have knowledge.**, **Consider someone who forms a true belief by guessing correctly; intuitively, this person does not know the proposition.**, **If the mere truth of a belief were enough for knowledge, any lucky guess would count as knowledge, though it clearly should not.**
