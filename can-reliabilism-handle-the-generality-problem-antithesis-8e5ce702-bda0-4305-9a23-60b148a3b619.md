# Can Reliabilism handle the generality problem?

**Node Type:** antithesis
**Node ID:** 8e5ce702-bda0-4305-9a23-60b148a3b619
**Depth:** 4
**Terminal:** No

**Parent:** [Reliabilism](reliabilism-synthesis-24b421a0-08f3-4c66-b0fb-2acc7f5da514.md)

## Content

**There is a plausible principle that the reliability of a process cannot be measured without specifying which 'type' of the process is being evaluated. This principle challenges reliabilism because it does not specify which level of generality should be used to classify a process as reliable.**, **Consider a case where one can describe a cognitive process at multiple levels of generality, such as 'vision' versus 'seeing in dim light'. Intuitively, we might see the reliability differing at these levels, complicating the claim that any instance of a process can simply count as reliable or not.**
