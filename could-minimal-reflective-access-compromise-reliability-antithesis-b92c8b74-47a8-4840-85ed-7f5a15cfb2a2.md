# Could minimal reflective access compromise reliability?

**Parent:** [Minimalist Internalism](minimalist-internalism-synthesis-c1aff540-8ee4-4dca-921e-260ac4a2c333.md)

## Content

**Consider a scenario where reflective access is minimal and leads to errors in judgment. Intuitively, lesser access might compromise the reliability associated with stronger reflective access.**, **Reliability in gaining true beliefs might be jeopardized when reflective access is kept minimal.**, **Minimal reflective access questions the reliability calibration if it allows unjustified true beliefs.**
