# Do all true beliefs automatically constitute knowledge regardless of context?

**Node Type:** antithesis
**Node ID:** ac97dde7-5e35-47e9-a6ed-bb260bf647c2
**Depth:** 4
**Terminal:** No

**Parent:** [Hybrid Contextual Invariantism](hybrid-contextual-invariantism-synthesis-adfbf27a-f780-498c-b828-265cba5bed4f.md)

## Content

**There is a plausible principle that knowledge requires justification beyond mere true belief. This principle together with beliefs based solely on context implies that not all true beliefs are knowledge.**, **Suppose we have a case where a belief is true in one context due to coincidence; intuitively, we must say it lacks knowledge.**
