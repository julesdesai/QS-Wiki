# Does the theory account for hallucinations where there is no corresponding external object?

**Node Type:** antithesis
**Node ID:** 512b203c-c34c-43e6-91dc-6b8f45d545f1
**Depth:** 4
**Terminal:** No

**Parent:** [Direct Realism with Strong Correspondence](direct-realism-with-strong-correspondence-synthesis-28884d49-4ced-45da-83a0-528139a02f57.md)

## Content

**Consider hallucinations where an individual has vivid perceptual experiences without any external object causing them, such as seeing pink elephants.**, **Intuitively, in hallucinations, there is no physical object corresponding to the perceptual experience.**, **If hallucinations exist, perceptual experiences cannot be assumed to always correspond to external objects.**
