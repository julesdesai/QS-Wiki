# Is all justified true belief a product of cognitive virtues?

**Node Type:** antithesis
**Node ID:** de9635c8-6c2c-4c4b-9833-360d9af48257
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-7681d6b8-18d3-4c19-95f1-d061b70d0c81.md)

## Content

**It is implausible that every justified true belief necessitates cognitive virtues when non-intellectual factors, such as memory or perception, function reliably outside voluntary control.**, **Justification as reliable belief production does not constitute active virtuous behavior, possibly excluding instances where truth is effortlessly achieved.**
