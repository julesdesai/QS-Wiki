# Is knowledge possible with unreliable memory sources?

**Node Type:** antithesis
**Node ID:** 6ead22f0-e672-4a83-8fb5-36b7de0f99f7
**Depth:** 4
**Terminal:** No

**Parent:** [Memory-Integrated Phenomenalism](memory-integrated-phenomenalism-synthesis-20b6ad46-c9b1-41b8-87a0-7619b03b72c5.md)

## Content

**Suppose a case where a person's memory frequently alters stored information, resulting in contradictions. Intuitively, such unreliable memory would not contribute positively to gaining knowledge.**, **It is implausible that knowledge can stem from unreliable memory, which undercuts the reliability needed for genuine knowledge.**
