# Can differing contexts lead to the same belief being justified and unjustified simultaneously?

**Node Type:** antithesis
**Node ID:** a3cc9b8d-4b60-4c61-9f7b-b00a0f1b8846
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism about Justification](contextualism-about-justification-synthesis-1dd27965-c99f-4b62-ae25-50f61d9bceba.md)

## Content

**Consider a situation where a belief is justified in one context but unjustified in another, with all facts about the belief itself remaining constant.**, **It is implausible that a belief can be both justified and unjustified at the same time because it violates the principle of non-contradiction, which holds that contradictory statements cannot both be true in the same sense.**
