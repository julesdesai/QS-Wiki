# Is it possible for a strongly coherent belief system to be based on a single falsehood?

**Node Type:** antithesis
**Node ID:** dbb8c59c-7fd5-4a86-8ceb-b0d4e5620339
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism: System of Beliefs](coherentism-system-of-beliefs-synthesis-446f5d76-49b9-4fbc-a49a-1aaf884652f2.md)

## Content

**A belief system can be structured around a central false belief yet remain internally coherent.**, **A central false belief undermines the justification that coherence supposedly provides for related beliefs.**, **Knowledge requires that beliefs be both true and justified.**, **Thus, even a strongly coherent belief system based on a single falsehood fails to confer knowledge.**
