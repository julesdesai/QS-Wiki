# Can context mistakenly identify alternatives?

**Node Type:** antithesis
**Node ID:** e025ee12-69e8-4111-b8e2-21c11e455bc2
**Depth:** 4
**Terminal:** No

**Parent:** [Relevant Alternatives Theory](relevant-alternatives-theory-synthesis-d7f52b7c-07b8-44a2-8804-2022a32ed27e.md)

## Content

**Suppose we have a case in which the context leads a person to consider false alternatives as relevant. Intuitively, we either have to say that their knowledge is impaired or that context can be mistaken. If we say their knowledge is impaired, then knowledge is unstable. If we say context can be mistaken, then context cannot fully determine relevance.**
