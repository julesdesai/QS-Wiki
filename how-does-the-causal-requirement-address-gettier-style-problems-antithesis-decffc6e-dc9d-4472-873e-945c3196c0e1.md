# How does the causal requirement address Gettier-style problems?

**Depth:** 4
**Terminal:** No

**Parent:** [Causal Theory of Knowledge](causal-theory-of-knowledge-synthesis-b3086f02-fe19-4187-b3b5-dcf882df82f4.md)

## Content

**Consider a situation wherein Smith has strong evidence for a proposition that turns out to be false, but due to a bizarre coincidence, he still forms a true belief through another unrelated element. Intuitively, we would judge Smith does not know the proposition.**, **Gettier-style problems suggest that a belief formed in a causally coincidental or lucky way—even if true—might fail to be knowledge, raising the question if the causal requirement sufficiently addresses these types of problems.**
