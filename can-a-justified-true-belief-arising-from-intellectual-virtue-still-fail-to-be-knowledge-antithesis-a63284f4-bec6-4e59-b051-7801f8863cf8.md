# Can a justified true belief arising from intellectual virtue still fail to be knowledge?

**Node Type:** antithesis
**Node ID:** a63284f4-bec6-4e59-b051-7801f8863cf8
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology Summary](virtue-epistemology-summary-synthesis-6b17452f-daf9-4e36-a5b5-c7d2a1d34c57.md)

## Content

**Consider a scenario in which a person forms a belief that is true and justified by reliable methods, such as memory or perception, yet the belief is true only due to random chance. Intuitively, we would judge that the person lacks knowledge.**, **There is a plausible principle that says knowledge cannot depend on luck. This principle together with the conception that justified true belief can arise from intellectual virtue entails that not all justified true beliefs are knowledge.**
