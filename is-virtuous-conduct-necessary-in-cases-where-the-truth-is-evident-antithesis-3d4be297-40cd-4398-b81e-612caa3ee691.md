# Is virtuous conduct necessary in cases where the truth is evident?

**Node Type:** antithesis
**Node ID:** 3d4be297-40cd-4398-b81e-612caa3ee691
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-7681d6b8-18d3-4c19-95f1-d061b70d0c81.md)

## Content

**Suppose we have a case where clear evidence is readily available, resulting in a true belief formed without manifest virtue. Intuitively, we would still attribute knowledge in such cases.**, **When external evidence is so strong, it minimizes the necessity for virtues like skepticism or diligence to achieve true belief, questioning if virtue is crucial in these instances.**
