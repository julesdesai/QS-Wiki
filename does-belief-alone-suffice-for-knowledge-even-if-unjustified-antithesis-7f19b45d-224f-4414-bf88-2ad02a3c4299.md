# Does belief alone suffice for knowledge, even if unjustified?

**Node Type:** antithesis
**Node ID:** 7f19b45d-224f-4414-bf88-2ad02a3c4299
**Depth:** 4
**Terminal:** No

**Parent:** [Finite Infinitism](finite-infinitism-synthesis-ec56b28f-9687-4f6e-b141-48dc2e075636.md)

## Content

**Consider a scenario where a person believes a false proposition. Intuitively, this does not count as knowledge.**, **If a belief alone suffices for knowledge, then false beliefs would incorrectly count as knowledge.**, **There is a plausible principle that for a belief to be knowledge, it must be justified, meaning belief alone is insufficient.**
