# Is proper function always necessary for knowledge if the belief is accidentally true?

**Node Type:** antithesis
**Node ID:** ba7b2abc-5971-4053-a4d1-bd0ab6c89ec8
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-66a6c079-b861-44cf-ac2d-ed1fae76b934.md)

## Content

**Suppose we have a case in which a person's cognitive faculties are impaired, and the person forms a belief that coincidentally turns out to be true. Intuitively, we would judge that the person does not have knowledge.**, **It is implausible that merely having a true belief in an environment conducive to truth suffices for knowledge without consideration of how that belief was formed.**
