# How does perceptual disagreement affect independent verification?

**Node Type:** antithesis
**Node ID:** 60dab5b4-63bf-436f-9407-788fd3a80adf
**Depth:** 4
**Terminal:** No

**Parent:** [Perception as Phenomenal-Independence](perception-as-phenomenal-independence-synthesis-76d6e707-cdc6-44d9-83d6-555caf4d40eb.md)

## Content

**Consider a scenario in which two individuals perceive the same event differently, such as one seeing a red object as purple. In such a scenario, intuitively we would judge that independent verification might yield contradictory results for both individuals.**, **There is a plausible principle that verification should resolve perceptual differences. This principle together with the need for independent verification implies that resolution of perception defers knowledge acquisition.**, **It is implausible that knowledge reliant on independent verification could clearly ascertain truth when competing perceptual interpretations arise.**
