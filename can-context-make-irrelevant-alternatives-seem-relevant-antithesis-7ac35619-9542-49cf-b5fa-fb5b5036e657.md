# Can context make irrelevant alternatives seem relevant?

**Node Type:** antithesis
**Node ID:** 7ac35619-9542-49cf-b5fa-fb5b5036e657
**Depth:** 4
**Terminal:** No

**Parent:** [Relevant Alternatives Theory](relevant-alternatives-theory-synthesis-d7f52b7c-07b8-44a2-8804-2022a32ed27e.md)

## Content

**Suppose we have a case where the context makes irrelevant alternatives seem relevant. Intuitively, we either have to say that knowledge is sensitive to errors or that context misguides our relevance assessment. If we say knowledge is error-sensitive, then knowledge is precarious. If we say context misguides, then not all context judgments about relevance are correct.**
