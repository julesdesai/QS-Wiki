# Can causally isolated agents or environments still generate justified beliefs?

**Node Type:** antithesis
**Node ID:** 16e545f9-83f6-4329-98e4-5908a837fbdc
**Depth:** 4
**Terminal:** No

**Parent:** [Goldman's Causal Theory](goldmans-causal-theory-synthesis-55eaa724-2453-46d2-8e70-b1e77bdcb753.md)

## Content

**Consider an agent living in a simulated environment where all inputs are artificial, yet the agent forms true beliefs about that environment.**, **In environments where there is no direct causal connection to the real world, the beliefs held can still be true and consistent.**, **A requirement for a causal connection to the actual external world is implausible in explaining knowledge in such cases.**
