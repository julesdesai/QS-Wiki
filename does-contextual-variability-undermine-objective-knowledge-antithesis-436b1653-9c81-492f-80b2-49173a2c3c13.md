# Does contextual variability undermine objective knowledge?

**Node Type:** antithesis
**Node ID:** 436b1653-9c81-492f-80b2-49173a2c3c13
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Infinitism](contextualist-infinitism-synthesis-74ebcf03-3de8-4e28-950c-dc2eefc9dc13.md)

## Content

**A case in which the context changes the sufficiency of justification for the same belief suggests variability.**, **If justification sufficiency depends heavily on context, then knowledge would lack an objective foundation.**, **It is implausible that knowledge could be sound if it fluctuates with contextual changes.**
