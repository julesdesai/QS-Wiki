# Is every instance of true belief from virtue non-accidental?

**Node Type:** antithesis
**Node ID:** 830f5567-fff1-4653-ac6f-f469c37adb98
**Depth:** 4
**Terminal:** No

**Parent:** [Anti-Luck Virtue Epistemology](anti-luck-virtue-epistemology-synthesis-fe5301ab-5101-4f93-ba1f-e195247489de.md)

## Content

**There is a plausible principle that a belief can be true due to intellectual virtue but still be mostly accidental or coincidental. This principle together with the claim that knowledge requires non-accidental truth implies a category of true beliefs that are intuitively knowledge but fail the non-accidental condition.**, **Consider a case in which a person's true belief aligns with a lucky event occurring. Even if the belief arises from intellectual virtue, the luck element challenges the belief's status as knowledge.**
