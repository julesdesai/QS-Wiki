# Can knowledge exist if all belief is influenced by luck?

**Node Type:** antithesis
**Node ID:** ce02c5c6-3caa-4376-a2ce-860e7a99d123
**Depth:** 4
**Terminal:** No

**Parent:** [Anti-Luck Virtue Epistemology](anti-luck-virtue-epistemology-synthesis-fe5301ab-5101-4f93-ba1f-e195247489de.md)

## Content

**It is implausible that any human belief can be completely isolated from luck, given the inherent unpredictability of the world.**, **Suppose we agree that all true beliefs are influenced in some way by luck. Intuitively, we would have to say that no belief is knowledge or that knowledge tolerates a degree of luck, challenging the claim that a belief must not be true by luck to count as knowledge.**
