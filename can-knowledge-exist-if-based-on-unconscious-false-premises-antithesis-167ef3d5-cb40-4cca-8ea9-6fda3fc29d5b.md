# Can knowledge exist if based on unconscious false premises?

**Node Type:** antithesis
**Node ID:** 167ef3d5-cb40-4cca-8ea9-6fda3fc29d5b
**Depth:** 4
**Terminal:** No

**Parent:** [No False Grounds Theory](no-false-grounds-theory-synthesis-37dfac32-afee-4ea8-9a67-23e7efbc2507.md)

## Content

**There is a plausible principle that knowledge should not be reliant on any false premises, regardless of whether these premises are consciously or subconsciously introduced.**, **Suppose we have a case where a person forms a belief through a subconscious process that is reliable but relies on a false premise. Intuitively, it seems that such a belief might still rely on false grounds.**
