# Is a belief formed by a usually reliable process still justified if the specific instance is unreliable?

**Node Type:** antithesis
**Node ID:** 09f395f4-1f53-454b-b462-2f8b0e1982e9
**Depth:** 4
**Terminal:** No

**Parent:** [Externalism with Reliabilism](externalism-with-reliabilism-synthesis-979d7449-6923-4b84-9ee7-21896810f66d.md)

## Content

**Consider a cognitive process that is generally reliable, but in a particular instance, it fails due to some unnoticed error. Intuitively, we would judge that the resulting belief is not justified in that instance**, **Knowledge requires that a belief be both true and reliably formed, questioning justification if the usual reliability falters in that instance**
