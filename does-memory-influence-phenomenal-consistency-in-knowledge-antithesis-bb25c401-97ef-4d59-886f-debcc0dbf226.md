# Does memory influence phenomenal consistency in knowledge?

**Node Type:** antithesis
**Node ID:** bb25c401-97ef-4d59-886f-debcc0dbf226
**Depth:** 4
**Terminal:** No

**Parent:** [Phenomenal Realism with Consistency Conditions](phenomenal-realism-with-consistency-conditions-synthesis-05f9226f-1221-4391-b2ae-790270c7a451.md)

## Content

**Consider a scenario in which a person recalls a phenomenon differently at two points in time due to memory distortion but still knows the proposition about it. Intuitively, changes in memory representation should not erase prior knowledge.**, **It is implausible that phenomenal consistency must be maintained through memory, since knowledge can persist despite memory's unreliability.**
