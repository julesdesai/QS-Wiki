# Can a belief be justified if it coheres with a false belief system?

**Node Type:** antithesis
**Node ID:** de225598-9f35-4c5c-a1f4-d19e4817b324
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism: System of Beliefs](coherentism-system-of-beliefs-synthesis-446f5d76-49b9-4fbc-a49a-1aaf884652f2.md)

## Content

**A coherent system of beliefs can include false beliefs.**, **If a belief system includes false beliefs, then it cannot provide adequate justification for other beliefs.**, **A person knows a proposition only if it is adequately justified.**, **Therefore, a belief that coheres with a false belief system is not adequately justified and does not constitute knowledge.**
