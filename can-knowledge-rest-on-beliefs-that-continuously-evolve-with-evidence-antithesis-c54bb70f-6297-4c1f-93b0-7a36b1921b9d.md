# Can knowledge rest on beliefs that continuously evolve with evidence?

**Node Type:** antithesis
**Node ID:** c54bb70f-6297-4c1f-93b0-7a36b1921b9d
**Depth:** 4
**Terminal:** No

**Parent:** [Dynamic Foundationalism](dynamic-foundationalism-synthesis-25d83013-c58a-42d7-a045-2c4c94be4d38.md)

## Content

**Knowledge implies a stable set of beliefs upon which it is based.**, **For a belief to serve as a foundation, it must resist frequent changes unless compelling evidence suggests otherwise.**, **If foundational beliefs constantly adapt, then the beliefs derived from them are never securely justified.**
