# Does coherence guarantee truth?

**Node Type:** antithesis
**Node ID:** e17e763f-40b5-4a9a-abba-5ddd8f18de02
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism with Justified Belief](coherentism-with-justified-belief-synthesis-01e10c3b-52a8-4097-b02c-3f010fc56355.md)

## Content

**There is a plausible principle that a coherent system of beliefs can be entirely false if the beliefs, though consistent and supporting each other, do not correspond to reality.**, **A system of beliefs in which all members of a community believe in and cohere with each other's false beliefs is still a system in which all the beliefs are false.**
