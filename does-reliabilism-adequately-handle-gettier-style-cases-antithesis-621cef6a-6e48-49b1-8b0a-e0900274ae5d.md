# Does reliabilism adequately handle Gettier-style cases?

**Node Type:** antithesis
**Node ID:** 621cef6a-6e48-49b1-8b0a-e0900274ae5d
**Depth:** 4
**Terminal:** No

**Parent:** [Reliabilism](reliabilism-synthesis-5d1929f3-45fe-4f02-bb7b-941a1c5017f0.md)

## Content

**Suppose a person forms a belief using a reliable process but has a false intermediate conclusion, as in a Gettier case, such as mistaking a realistic hologram for a real object. Intuitively, the person does not have knowledge because the belief is based on a false premise.**, **It is implausible for a belief to be considered knowledge when it is merely coincidentally true due to an error in reasoning.**
