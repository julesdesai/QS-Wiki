# How does this view handle Gettier-style problems?

## Content

**Consider a scenario in which a person forms a true belief based on false premises, such as believing a stopped clock is accurate and thereby getting the correct time by chance. Intuitively, we would judge this is not knowledge**, **Gettier-style cases show that justified true belief is not sufficient for knowledge because these beliefs can still result from luck, which is incompatible with the common understanding of knowledge**
