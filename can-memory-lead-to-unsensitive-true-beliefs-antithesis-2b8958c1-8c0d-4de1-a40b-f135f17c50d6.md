# Can memory lead to unsensitive true beliefs?

**Node Type:** antithesis
**Node ID:** 2b8958c1-8c0d-4de1-a40b-f135f17c50d6
**Depth:** 4
**Terminal:** No

**Parent:** [Sensitive belief theory of knowledge](sensitive-belief-theory-of-knowledge-synthesis-a04ea135-b51c-4804-8a79-c6b4af4a4910.md)

## Content

**Imagine a situation in which a person recalls a true historical fact from memory, but this memory was formed randomly and had it been stored slightly differently, the person would recall something false. Intuitively, we would question whether the person truly knows the fact despite remembering it correctly.**, **When a person's true beliefs about past events result from unreliable memory processes, their belief lacks sensitivity, problematizing the claim that they possess knowledge.**
