# Should true belief be considered knowledge if based on unreliable foundations?

**Node Type:** antithesis
**Node ID:** fc8bb344-aa12-45a1-a0c9-c3176d32db24
**Depth:** 4
**Terminal:** No

**Parent:** [Fallibilist Foundationalism](fallibilist-foundationalism-synthesis-ed3484e7-a0b3-47a4-939c-ec83c0f7bb7c.md)

## Content

**Consider a situation where someone guesses the winner of a horse race and happens to be right, but based on flawed reasoning and lack of evidence. Intuitively, we would judge that this true belief does not count as knowledge.**, **There is a plausible principle that knowledge must exclude beliefs that are true merely by chance or luck, which suggests that true belief does not constitute knowledge if based on unreliable foundations.**
