# If a belief is true and virtuous but lacks understanding, is it knowledge?

**Node Type:** antithesis
**Node ID:** d6890df7-018b-4b5c-9e5e-a641f75ec8c8
**Depth:** 4
**Terminal:** No

**Parent:** [Enhanced Virtue Reliabilism with Error Avoidance](enhanced-virtue-reliabilism-with-error-avoidance-synthesis-bae843f7-47b1-4791-af4f-a08b903c892d.md)

## Content

**Consider a scenario where someone has a true belief through reliable means yet lacks any understanding of why the belief is true. Intuitively, we would judge that this does not constitute full knowledge.**, **There is a plausible principle that knowledge involves understanding the truth, not just holding a true belief, which suggests that mere reliability without understanding is inadequate.**
