# Can a belief be safe if nearby errors are possible?

**Node Type:** antithesis
**Node ID:** 5cd5db44-de4e-4a9d-8beb-c0f3dc154ca4

## Content

**A belief is safe if in nearby possible situations it remains true.**, **There is a case where a person’s belief is true, but if minor conditions change slightly, the belief would become false.**, **If a belief can easily become false with minor variations in circumstances, it is questionable whether this belief is genuinely safe.**
