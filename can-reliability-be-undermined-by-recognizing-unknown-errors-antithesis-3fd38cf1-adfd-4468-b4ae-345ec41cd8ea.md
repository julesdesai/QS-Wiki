# Can reliability be undermined by recognizing unknown errors?

**Node Type:** antithesis
**Node ID:** 3fd38cf1-adfd-4468-b4ae-345ec41cd8ea
**Depth:** 4
**Terminal:** No

**Parent:** [Process Reliabilism](process-reliabilism-synthesis-094c2ddc-bd9c-46c2-87b1-e486aaff8cb4.md)

## Content

**Consider a scenario where a cognitive process produces true beliefs most of the time. In this scenario, if the process produces false beliefs occasionally without the subject's knowledge, it is questionable whether this process is reliable.**, **A plausible principle holds that for a cognitive process to be reliable, one must have some awareness of its failure rate. If there is no awareness of errors, the belief possibly lacks reliability.**
