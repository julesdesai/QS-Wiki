# Do indirect perceptual experiences provide any certainty about external objects?

**Node Type:** antithesis
**Node ID:** 5850f3a3-5772-4c4c-abab-f2083fc54506
**Depth:** 4
**Terminal:** No

**Parent:** [Indirect Realism](indirect-realism-synthesis-197f3805-c8e1-415f-87ee-9ac547e71c62.md)

## Content

**It is implausible that experiences which only indirectly present objects can provide certainty about them given they can be deceptive or incomplete.**, **Suppose we have a case in which a person observes a complex painting through a smudged glass. Intuitively, we either have to say that the perception through the glass is affected or unaffected. If we say it is unaffected, then all perceptions should be equally trustworthy, which is unreasonable. If we say it is affected, then certainty about the external object is compromised.**
