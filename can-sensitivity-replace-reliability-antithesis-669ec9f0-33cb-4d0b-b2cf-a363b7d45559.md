# Can sensitivity replace reliability?

**Parent:** [Reliabilism](reliabilism-synthesis-27039b77-724d-4113-8b7b-b992ae688652.md)

## Content

**Consider a scenario where a belief matches reality but would not have if conditions slightly changed.**, **Intuitively, if the belief is not sensitive to slight changes, it seems insufficient to count as knowledge, challenging the sufficiency of reliability.**
