# Can knowledge rest solely on true belief according to Contextual Foundationalism?

## Content

**A case in which an individual believes a true proposition simply because they guessed is not a case of knowledge.**, **There is a plausible principle that knowledge requires more than just true belief. This principle together with the example entails that true belief might be insufficient for knowledge.**
