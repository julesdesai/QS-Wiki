# Can context-dependent knowledge resolve issues of memory reliability?

**Parent:** [Contextualism](contextualism-synthesis-1e9d0e24-887b-4fad-bf5c-71b82495d573.md)

## Content

**Suppose that a person's context-sensitivity leads them to accept or reject past knowledge claims depending on current contextual standards. Intuitively, this threatens the reliability and coherence of one’s memory-based knowledge.**, **There is a plausible principle that memory should provide a stable and coherent foundation for knowledge, yet the contextual shifts seem to destabilize this coherence.**
