# How can a belief be knowledge if it lacks justified true belief?

**Node Type:** antithesis
**Node ID:** 7f9f356a-741c-41de-9d2b-490f32be7aba
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Externalism](epistemic-externalism-synthesis-a9c372ef-1ae7-460b-b26c-8a82dc6a1ee1.md)

## Content

**There is a plausible principle that knowledge requires justified true belief.**, **A belief can be true and meet external conditions but still lack justification from the agent's perspective.**, **Intuitively, a belief that is true by luck rather than through justified processes is not knowledge.**
