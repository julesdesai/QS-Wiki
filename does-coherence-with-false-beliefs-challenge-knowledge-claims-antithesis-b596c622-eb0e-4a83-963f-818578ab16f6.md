## Content

**It is plausible that a justification system should not be based on false beliefs. If a set of beliefs is coherent but includes false elements, then the knowledge claim is undermined.**, **Consider a case in which a person has a coherent set of beliefs, but one foundational belief is false, intuitively we would judge this person does not have knowledge.**
