# Is memory an unreliable basis for knowledge about the external world?

**Node Type:** antithesis
**Node ID:** 87702cfe-28b6-49c5-9d15-6be388a5c5cc
**Depth:** 4
**Terminal:** No

**Parent:** [Phenomenalism with Appeal to Memory](phenomenalism-with-appeal-to-memory-synthesis-2b7704c1-89ef-4cc3-a1d9-427b8175dc3f.md)

## Content

**Memory is susceptible to distortion and bias, which can lead to false recollections of past experiences.**, **If past perceptual experiences stored in memory are unreliable, then the inferences made about external objects based on these experiences are also unreliable.**
