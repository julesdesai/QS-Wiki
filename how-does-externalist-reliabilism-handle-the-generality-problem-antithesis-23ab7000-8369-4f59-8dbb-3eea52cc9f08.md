# How does externalist reliabilism handle the generality problem?

**Node Type:** antithesis
**Node ID:** 23ab7000-8369-4f59-8dbb-3eea52cc9f08
**Depth:** 4
**Terminal:** No

**Parent:** [Externalist Reliabilism](externalist-reliabilism-synthesis-a51e876d-808b-45d8-97dd-3bdec299c925.md)

## Content

**There is a plausible principle that for a process to be deemed reliable, it must be identified in a way that is neither too specific nor too general, allowing for appropriate classification.**, **It is implausible to assume that the same process classification applies universally across cases without additional criteria for context-sensitive identification.**
