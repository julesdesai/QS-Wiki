# Does reliability entail infallibility for knowledge?

**Node Type:** antithesis
**Node ID:** e474dee8-c7d5-416d-b003-76d8f92f0679
**Depth:** 4
**Terminal:** No

**Parent:** [Process Reliabilism](process-reliabilism-synthesis-383ba069-e0f1-4291-bdc7-eacefc701040.md)

## Content

**Suppose we have a case in which an expert consistently provides correct information except for rare errors. Intuitively, if a process is reliable, occasional errors should not disqualify it entirely. If reliability implies complete infallibility, then almost all human processes would be unreliable.**, **It is implausible that any human cognitive process is completely infallible, yet people are generally considered to know many things. If infallibility is required for reliability, then knowledge becomes too stringent.**
