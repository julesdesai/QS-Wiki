# Does a fulfilled cognitive task always align with truth?

**Node Type:** antithesis
**Node ID:** e713085a-f63b-4e41-8a5d-6958717e9cb0
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Functionalism with Cognitive Task Fulfillment](epistemic-functionalism-with-cognitive-task-fulfillment-synthesis-3edd3165-e9fd-4e05-81df-6b63dbbb89eb.md)

## Content

**Consider a scenario where a person completes a reasoning task leading to a false conclusion because a generally reliable source provided faulty data. Intuitively, this does not seem like knowledge even though the task was executed correctly.**, **A plausible principle is that truth is a criterion for knowledge, implying task fulfillment with a false outcome does not equate to knowing.**
