# Does context allow arbitrary beliefs to count as knowledge?

**Node Type:** antithesis
**Node ID:** 20c98344-29a4-499f-86d4-a990705f667f
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Foundationalism](contextualist-foundationalism-synthesis-9835edc1-dd3b-4e9d-81fa-8a065de25b95.md)

## Content

**Suppose a belief is considered foundational and justified because of peculiar contextual features, like cultural norms, which can vary wildly. Intuitively, this allows almost any belief to be potentially justified, diluting the notion of knowledge.**
