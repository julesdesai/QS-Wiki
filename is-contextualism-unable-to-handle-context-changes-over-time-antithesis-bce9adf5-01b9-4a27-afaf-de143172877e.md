# Is contextualism unable to handle context changes over time?

**Node Type:** antithesis
**Node ID:** bce9adf5-01b9-4a27-afaf-de143172877e
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism about Knowledge](contextualism-about-knowledge-synthesis-3f771a9e-26d6-43c4-b604-a31f8bd9947c.md)

## Content

**Knowledge claims made at one time and context could later be assessed differently as contexts change.**, **The same proposition believed to be true and justified in one context might not be justified in another due to changes in standards.**, **It is troubling if knowledge is so fluid that it can be lost or gained simply through contextual shifts over time.**
