# Is proper cognitive function always required for justification?

**Node Type:** antithesis
**Node ID:** d89d0f69-b11f-44dc-ad11-64789a259ed1
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology Summary](virtue-epistemology-summary-synthesis-6b17452f-daf9-4e36-a5b5-c7d2a1d34c57.md)

## Content

**Consider a scenario where a person suffering from minor memory lapses still retains knowledge that they experienced. Intuitively, such scenarios suggest knowledge persists despite imperfect cognitive functioning.**, **The principle that justification involves proper cognitive faculties functioning suggests that such justification is not necessary for knowledge.**
