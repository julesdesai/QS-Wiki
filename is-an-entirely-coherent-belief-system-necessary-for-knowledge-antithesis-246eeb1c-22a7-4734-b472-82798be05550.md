# Is an entirely coherent belief system necessary for knowledge?

**Node Type:** antithesis
**Node ID:** 246eeb1c-22a7-4734-b472-82798be05550
**Depth:** 4
**Terminal:** No

**Parent:** [Externalist Coherentism](externalist-coherentism-synthesis-f4ef4378-ebfc-4a41-8845-189c2c8e0f20.md)

## Content

**A person can possess knowledge with some incoherent beliefs as long as the relevant belief is true and justified.**, **Perfect coherence in a belief system is a stringent requirement that is unlikely achievable for any finite human knower.**, **Beliefs can be justified without full coherence, particularly in isolated domains where conflicting beliefs in other areas don't undermine the justification.**
