# Does the need for a fact undermine externalist claims?

**Node Type:** antithesis
**Node ID:** e597ece8-e391-491a-8cc6-0c57cf30d306
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Externalism](epistemic-externalism-synthesis-539989cd-f86f-4268-aeb1-80f5fff257d5.md)

## Content

**Suppose we have a case in which a person has a belief that perfectly matches a currently unknown fact. Intuitively, we must either accept that this belief counts as knowledge or restrict knowledge to known facts. If we accept it as knowledge, then unknowable connections to facts seem to suffice. If we restrict knowledge, then falsehoods about unknowns challenge reliable process assumptions.**
