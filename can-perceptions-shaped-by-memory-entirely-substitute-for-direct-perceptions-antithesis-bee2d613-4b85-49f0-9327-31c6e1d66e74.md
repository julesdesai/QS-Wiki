# Can perceptions shaped by memory entirely substitute for direct perceptions?

**Node Type:** antithesis
**Node ID:** bee2d613-4b85-49f0-9327-31c6e1d66e74
**Depth:** 4
**Terminal:** No

**Parent:** [Revised Phenomenalism Allowing Memory Role](revised-phenomenalism-allowing-memory-role-synthesis-5fb5abbe-6adb-4855-a476-25c8a91d11a9.md)

## Content

**External objects are typically understood as providing a check on perceptions. This idea together with the claim that external objects can't be known independently of perceptions and memory features leads to the conclusion that memory-influenced perceptual experiences could lack external validation, reducing their claim to actual knowledge.**, **Consider a situation where someone “remembers” an object existing due to repeated discussions about it, despite never having encountered it directly. Intuitively, without direct perception or interaction, their knowledge appears suspect.**
