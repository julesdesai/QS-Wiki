# Can external objects be entirely reduced to perceptual experiences?

**Node Type:** antithesis
**Node ID:** ffd25d97-c726-4c29-b6f0-317049af9401
**Depth:** 4
**Terminal:** No

**Parent:** [Objective Phenomenalism](objective-phenomenalism-synthesis-91c0e065-6d6c-4b9a-9519-a161a622cd2d.md)

## Content

**There is a plausible principle that the continuity and persistence of external objects extend beyond any particular set of perceptual experiences. This principle suggests that objects can exist without being perceived at a given time by anyone.**, **Consider a scenario in which a tree falls in a forest with no one to hear it. Intuitively, the tree still makes a sound, indicating external objects have properties independent of perception.**
