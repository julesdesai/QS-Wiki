# How does epistemic context variation challenge universal truth?

**Node Type:** antithesis
**Node ID:** 988b5477-1e82-4b94-812e-7ec0a3442dec
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism with Dispositional Belief](contextualism-with-dispositional-belief-synthesis-1dbb9d87-3561-45d5-9c36-839374bee7ec.md)

## Content

**The idea that knowledge depends on the knower's context can imply that what is known as true in one context may not be true in another.**, **This raises the question of whether there can be universal truths, as intuitively, truth as a baseline should not vary substantially with context.**
