# Do cognitive constraints limit responsibility?

**Node Type:** antithesis
**Node ID:** 37074d26-466a-48e5-b572-5e11ab58543d
**Depth:** 4
**Terminal:** No

**Parent:** [Responsibilist Virtue Epistemology](responsibilist-virtue-epistemology-synthesis-d65854c6-8623-4bd4-aed3-f2a4bfae69c9.md)

## Content

**Cognitive constraints, such as memory or processing limitations, can affect belief formation.**, **A person might be as responsible as possible within these constraints but still form false beliefs.**, **This challenges whether such effort-limited beliefs count as knowledge.**
