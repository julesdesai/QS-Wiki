# Is it possible for true belief without justification to count as knowledge?

**Node Type:** antithesis
**Node ID:** 93cf47ea-2c46-4f86-894f-e4ed4a8f8680
**Depth:** 4
**Terminal:** No

**Parent:** [Evidential Externalism](evidential-externalism-synthesis-1f3be30d-cfb3-46d2-9d1a-294e056d7ea0.md)

## Content

**Suppose we have a case in which a person believes a proposition that happens to be true purely by chance, without any justification. Intuitively, we would have to say that the person merely got lucky rather than possessing genuine knowledge.**, **The presence of justification that connects belief to truth seems integral for transforming true belief into knowledge, entailing that true belief alone should not suffice for knowledge.**
