# Is it possible to have justified beliefs without true propositions?

**Node Type:** antithesis
**Node ID:** 3dbcbcca-c54b-4d36-bcf6-5a670c3ab8b3
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-ea617099-7596-484d-aca6-1b318b0033ac.md)

## Content

**Consider a scenario in which an individual's cognitive faculties lead them to believe a well-supported scientific theory that is later proven false. Intuitively, this belief seems justified despite failing the truth criterion.**, **There is a plausible principle that justification can occur independently of truth, challenging the view that a true proposition is necessary for justified belief.**
