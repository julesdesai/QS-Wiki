# Does relativism permit meaningful critique of oppressive beliefs or practices within a framework?

**Node Type:** antithesis
**Node ID:** 3441b01b-238a-45bb-b94f-0b8b10964ab8
**Depth:** 4
**Terminal:** No

**Parent:** [Relativism Summary](relativism-summary-synthesis-2665e075-79d4-4adb-9371-55cb5a5794f8.md)

## Content

**Consider a case in which a person's epistemic framework supports oppressive practices. Intuitively, these practices can still be critiqued from a moral standpoint.**, **A plausible principle is that critique requires some objective point of reference, which relativism denies between frameworks.**
