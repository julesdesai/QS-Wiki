# Is belief really optional for knowledge if belief seems essential?

**Node Type:** antithesis
**Node ID:** 8f0d0dee-bf38-4b9a-a700-24c6671751fc
**Depth:** 4
**Terminal:** No

**Parent:** [No-Belief Necessary View](no-belief-necessary-view-synthesis-562c34c9-8bb3-4b6d-bbdd-b0bc8f277594.md)

## Content

**A case in which a person sees a cat on the mat, has all evidence and reason to support this proposition, yet does not believe that the cat is on the mat does not intuitively count as knowledge under ordinary circumstances.**, **In ordinary language, if someone claims to know without belief, it seems contradictory to typical usage of "know."**
