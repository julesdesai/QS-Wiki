# Can malfunctioning cognitive faculties yield true beliefs that don't count as knowledge?

**Node Type:** antithesis
**Node ID:** 6b298924-f907-4f79-8bd1-6c97ca0624a1
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-eb8cad20-b2ac-4ebe-8b63-c23141145814.md)

## Content

**Consider a scenario where a person has a cognitive faculty malfunctioning but accidentally produces a true belief. Intuitively, we would judge that this is not a case of knowledge.**, **A person with a cognitive faculty malfunctioning could still arrive at a true belief, which creates a tension with the idea that functioning faculties are necessary for knowledge.**
