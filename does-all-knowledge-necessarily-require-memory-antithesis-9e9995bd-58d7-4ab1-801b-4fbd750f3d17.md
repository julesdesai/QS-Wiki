# Does all knowledge necessarily require memory?

**Node Type:** antithesis
**Node ID:** 9e9995bd-58d7-4ab1-801b-4fbd750f3d17
**Depth:** 4
**Terminal:** No

**Parent:** [Memory-Integrated Phenomenalism](memory-integrated-phenomenalism-synthesis-20b6ad46-c9b1-41b8-87a0-7619b03b72c5.md)

## Content

**Suppose a case where a person comes to know some mathematical fact through direct insight or intuition without relying on memory. Intuitively, this is a case of knowledge that does not originate from perceptual experiences or memory integration.**, **It is implausible that knowledge necessarily depends on memory if some knowledge can seemingly emerge from non-perceptual insights.**
