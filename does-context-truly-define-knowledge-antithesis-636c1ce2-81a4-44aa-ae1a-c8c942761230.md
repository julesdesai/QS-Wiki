# Does context truly define knowledge?

**Node Type:** antithesis
**Node ID:** 636c1ce2-81a4-44aa-ae1a-c8c942761230
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Foundationalism](contextualist-foundationalism-synthesis-b757e054-5c50-4073-a4ee-c1ee757cbdbb.md)

## Content

**A case in which a person believes they know it is going to rain because a weather app suggests a high chance of rain is not a case where the person knows it will rain if the app's data is faulty.**, **If foundational beliefs vary with context, then the reliability of knowledge claims becomes unstable across different contexts.**, **It is implausible that true belief justified in one context but not in another still constitutes knowledge.**
