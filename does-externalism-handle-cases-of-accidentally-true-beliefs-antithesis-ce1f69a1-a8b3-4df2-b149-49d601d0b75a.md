# Does externalism handle cases of accidentally true beliefs?

**Node Type:** antithesis
**Node ID:** ce1f69a1-a8b3-4df2-b149-49d601d0b75a
**Depth:** 4
**Terminal:** No

**Parent:** [Externalism](externalism-synthesis-5b44df35-6663-4278-91dc-65b3fcbf870c.md)

## Content

**Consider a scenario where a person forms a belief that it is raining by looking outside through a window but the window is painted to look like it's raining. Intuitively, we might judge that the person does not know it is raining, even if it actually is.**, **A true belief produced by a reliable process should not result in knowledge if the reliability is not connected to the actual belief formation.**
