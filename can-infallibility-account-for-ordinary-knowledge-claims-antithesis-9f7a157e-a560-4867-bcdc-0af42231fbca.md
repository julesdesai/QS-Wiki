# Can infallibility account for ordinary knowledge claims?

**Node Type:** antithesis
**Node ID:** 9f7a157e-a560-4867-bcdc-0af42231fbca
**Depth:** 4
**Terminal:** No

**Parent:** [Knowledge as Infallibility](knowledge-as-infallibility-synthesis-8f0bcd35-bb4c-4192-816d-76a63a41903c.md)

## Content

**Consider an ordinary scenario where a person correctly believes it will rain tomorrow based on a reliable weather report.**, **It is possible for the report to be incorrect, yet intuitively we would still say the person has knowledge if it does rain.**, **The requirement for no possibility of error makes it implausible to attribute knowledge in such common situations.**
