# Can cognitive malfunction still lead to true belief?

**Node Type:** antithesis
**Node ID:** cc8c9576-db93-4a6b-b748-1347b406fbd8
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Functionalism](proper-functionalism-synthesis-60a6689e-3dfa-4e29-ad7b-61cc33b34e1b.md)

## Content

**A person with cognitive faculties that occasionally malfunction may still form true beliefs in some instances.**, **There are cases where despite unreliable cognitive faculties, an individual seems to have knowledge, as when an intuitive guess turns out correct.**, **It challenges whether proper functioning is strictly necessary for knowledge if true belief can arise from malfunctioning faculties.**
