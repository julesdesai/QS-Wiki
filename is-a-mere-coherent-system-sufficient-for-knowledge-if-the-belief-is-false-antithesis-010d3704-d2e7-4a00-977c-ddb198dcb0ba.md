# Is a mere coherent system sufficient for knowledge if the belief is false?

**Node Type:** antithesis
**Node ID:** 010d3704-d2e7-4a00-977c-ddb198dcb0ba
**Depth:** 4
**Terminal:** No

**Parent:** [Hybrid Coherentist-Internalist View](hybrid-coherentist-internalist-view-synthesis-ffa44d27-7377-4f6d-8451-515975b92108.md)

## Content

**Consider a scenario in which a person has a coherent system of beliefs that is entirely false. In this scenario, intuitively, we would judge that the person does not have knowledge.**, **A belief system's coherence is not sufficient to constitute knowledge if the foundational beliefs within the system are false.**
