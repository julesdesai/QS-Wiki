# Does all knowledge stem from perceptual and memory causes?

**Node Type:** antithesis
**Node ID:** d4fd9ee6-de54-4119-b139-8543a3c8fbd9
**Depth:** 4
**Terminal:** No

**Parent:** [Memory-Integrated Phenomenalism](memory-integrated-phenomenalism-synthesis-20b6ad46-c9b1-41b8-87a0-7619b03b72c5.md)

## Content

**Consider the case of innate knowledge, such as an inherent understanding of basic mathematical principles. Intuitively, this knowledge does not arise from perceptual experiences or memory integration.**, **It is implausible that all knowledge is the effect of perception and memory if some knowledge appears to be innate.**
