# How does one differentiate changes in context from fluctuations in belief?

**Node Type:** antithesis
**Node ID:** 118e1589-3320-4362-83f9-f18bb940c5d6
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Contextualism Summary](epistemic-contextualism-summary-synthesis-463c65ed-13fd-4c3b-8fbb-fc52a847cb71.md)

## Content

**A person's belief in a proposition varies over time without any change in evidence.**, **Intuitively, one should not claim that context changes explain shifts in knowledge claims, undermining contextual reliance.**
