# How does the causal theory deal with the problem of deviant causal chains?

**Node Type:** antithesis
**Node ID:** 5534dcd1-d811-46a9-80ff-54ae7f7bf5db
**Depth:** 4
**Terminal:** No

**Parent:** [Causal Theory of Knowledge](causal-theory-of-knowledge-synthesis-dc61c008-27ed-4921-863e-badc9b961db6.md)

## Content

**Consider a scenario in which a burglar inadvertently causes someone to believe the truth, such as when a broken fence lets them see a rare bird, intuitively we would judge that despite the true belief, this is not knowledge.**, **There is a plausible principle that true belief must not only be causally connected but also caused in a non-deviant manner, to consider it knowledge.**
