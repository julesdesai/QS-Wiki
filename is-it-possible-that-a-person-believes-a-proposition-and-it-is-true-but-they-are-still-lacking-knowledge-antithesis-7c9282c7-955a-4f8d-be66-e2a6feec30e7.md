# Is it possible that a person believes a proposition and it is true but they are still lacking knowledge?

**Node Type:** antithesis
**Node ID:** 7c9282c7-955a-4f8d-be66-e2a6feec30e7
**Depth:** 4
**Terminal:** No

**Parent:** [Moderate Internalism](moderate-internalism-synthesis-80f7ec1a-00e0-477d-b8d6-7e46a6205c79.md)

## Content

**A person can believe something true but if the truth of the proposition is coincidental, it is less clear that we would want to call this knowledge**, **If a person believes a proposition without good reasons, especially if they are unaware of those reasons, their justified belief is in question**, **A belief based on misleading evidence happening to be true lacks an element that we intuitively expect from knowledge**
