# Can coherentism allow for isolated, contextually bound knowledge?

**Node Type:** antithesis
**Node ID:** 654d77c0-0006-4c39-840b-26b25436dcfe
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism Summary](coherentism-summary-synthesis-7bdcae0a-f633-4c38-9857-77d6ba49435c.md)

## Content

**Suppose we have a case involving a person who lives entirely alone on an island with beliefs about the world that are internally consistent but unconnected to external reality. Intuitively, we either have to say that this person lacks knowledge about the world or that knowledge is possible without real-world correspondence. If we say this person lacks knowledge, then coherence seems insufficient. If we say knowledge is possible without correspondence, then intuition about reality contradicts.**
