# Can false propositions generate knowledge?

**Node Type:** antithesis
**Node ID:** 8806c593-8cb9-4dda-80ba-d02f765b9087
**Depth:** 4
**Terminal:** No

**Parent:** [Relativism about Knowledge](relativism-about-knowledge-synthesis-db569ddc-fb91-431d-962c-b8c6e0bc3b22.md)

## Content

**A false proposition contradicts reality. If one accepts that false propositions can be known, then knowledge would not require a reality-consistent foundation, which seems implausible.**, **A person knows a proposition if it is true, hence false propositions should not be knowable.**
