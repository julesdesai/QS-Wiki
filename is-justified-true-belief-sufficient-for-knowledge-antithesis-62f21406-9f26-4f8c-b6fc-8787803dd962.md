# Is justified true belief sufficient for knowledge?

**Node Type:** antithesis
**Node ID:** 62f21406-9f26-4f8c-b6fc-8787803dd962
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism: Contextual Standards](contextualism-contextual-standards-synthesis-864b880e-c24e-4de1-9edf-fbd584c75f88.md)

## Content

**A case in which some person P believes that a stopped clock shows the correct time, and it is in fact the correct time, is not a case in which P knows the time.**, **A case in which some person P believes that a stopped clock shows the correct time, and it is in fact the correct time, is a case in which P has justification for their belief.**
