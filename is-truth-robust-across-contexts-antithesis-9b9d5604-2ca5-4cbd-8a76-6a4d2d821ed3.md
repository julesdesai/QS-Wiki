# Is truth robust across contexts?

**Node Type:** antithesis
**Node ID:** 9b9d5604-2ca5-4cbd-8a76-6a4d2d821ed3
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism with Reliabilism](contextualism-with-reliabilism-synthesis-c7c3d2a5-9a25-4e3f-9e1f-65010441e024.md)

## Content

**The requirement that a proposition must be true for a person to know it suggests that truth is a stable and fixed condition.**, **If knowledge standards vary with context but truth does not, then truth provides a stable benchmark, which conflicts with the variability suggested by contextual knowledge standards.**
