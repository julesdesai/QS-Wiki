# Is it problematic if changes in context can lead to an attribution of both knowledge and ignorance simultaneously?

**Node Type:** antithesis
**Node ID:** 772aa27c-cad4-4154-b377-8a422c7b96c1
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Contextualism Summary](epistemic-contextualism-summary-synthesis-463c65ed-13fd-4c3b-8fbb-fc52a847cb71.md)

## Content

**Suppose that in one context, a statement is taken as known, while in another it is not.**, **This would imply both knowledge and ignorance for the same statement, which is contradictory.**
