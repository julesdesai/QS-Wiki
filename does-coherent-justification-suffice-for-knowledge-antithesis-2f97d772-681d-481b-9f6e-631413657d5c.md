# Does coherent justification suffice for knowledge?

**Node Type:** antithesis
**Node ID:** 2f97d772-681d-481b-9f6e-631413657d5c
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism focuses on systematized justification](coherentism-focuses-on-systematized-justification-synthesis-825f84fd-be44-4c37-9d4c-4b4f5d772a40.md)

## Content

**It is implausible that coherence alone can justify a belief without reference to external facts.**, **A case in which a person has a coherent set of beliefs about the world due to being systematically misled is not a case in which that person has knowledge.**, **There is a plausible principle that knowledge requires a connection to the truth in a non-coincidental way. This principle together with the idea of coherent justification fails if coherence is all that is needed.**
