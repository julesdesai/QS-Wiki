# Can foundational beliefs themselves be unjustified?

**Node Type:** antithesis
**Node ID:** 19c80f18-2ba0-4762-b4ba-9ce461895e8a
**Depth:** 4
**Terminal:** No

**Parent:** [Foundationalism as an alternative to Coherentism](foundationalism-as-an-alternative-to-coherentism-synthesis-459d7634-bf33-408f-b245-ac7a68c56f44.md)

## Content

**There is a plausible principle that any belief, including a basic belief, can be unjustified if it is not supported by some evidence or reason.**, **Suppose we have a case where a person holds a basic belief that is self-evident to them, but others with the same information do not find it self-evident. Intuitively, the basic belief might not be justified without wider agreement or supporting evidence.**
