# Can context-based justification allow for arbitrary knowledge attributions?

**Node Type:** antithesis
**Node ID:** 17eb141a-774b-4e0f-80bb-8d45b6245147
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Theory of Knowledge](contextualist-theory-of-knowledge-synthesis-fc8174e2-112e-4e1b-9915-e4d5070db211.md)

## Content

**Consider a case in which a person's belief is labeled as knowledge because the context permits low standards for justification, such as anecdotal evidence. Intuitively, this feels arbitrary if justification standards can be unduly lax.**, **Suppose a scenario where storytelling suffices as knowledge about historical facts in one community context. It seems arbitrary to grant the title of knowledge without rigorous justification, raising doubts about epistemic reliability.**
