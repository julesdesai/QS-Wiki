# Must true belief always result from virtuous cognitive processes to count as knowledge?

**Node Type:** antithesis
**Node ID:** 1d6691dc-3df3-4074-b725-9c94b7063526
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-7681d6b8-18d3-4c19-95f1-d061b70d0c81.md)

## Content

**Suppose we have a case in which a scientist stumbles upon a significant discovery purely by accident, without using any cognitive capacity traditionally considered virtuous. Intuitively, it seems right to attribute knowledge to this scientist.**, **There is a plausible principle that knowledge requires true belief formed by some reliable process, but this process need not necessarily involve recognized cognitive virtues like attentiveness.**
