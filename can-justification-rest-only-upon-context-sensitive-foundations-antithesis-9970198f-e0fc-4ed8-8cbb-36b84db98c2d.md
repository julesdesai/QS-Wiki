# Can justification rest only upon context-sensitive foundations?

**Node Type:** antithesis
**Node ID:** 9970198f-e0fc-4ed8-8cbb-36b84db98c2d
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Foundationalism](contextualist-foundationalism-synthesis-9835edc1-dd3b-4e9d-81fa-8a065de25b95.md)

## Content

**It is implausible that only context-sensitive beliefs can ground justification because this could result in contradictory justified beliefs.**, **Consider a case where two equally plausible contexts justify two contradictory beliefs. Intuitively, allowing both to be justified without a more stable foundation poses a problem.**
