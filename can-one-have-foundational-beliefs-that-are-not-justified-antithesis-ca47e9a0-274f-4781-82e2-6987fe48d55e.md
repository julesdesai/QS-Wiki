# Can one have foundational beliefs that are not justified?

**Node Type:** antithesis
**Node ID:** ca47e9a0-274f-4781-82e2-6987fe48d55e
**Depth:** 4
**Terminal:** No

**Parent:** [Epistemic Foundationalism](epistemic-foundationalism-synthesis-e90eff9d-4b30-41a2-b4e3-4bbc38385542.md)

## Content

**Consider a case in which an individual has a foundational belief formed by unreliable means, like a random guess. Intuitively, we would judge that this belief is not justified despite being foundational.**, **It is implausible that a belief can be epistemically secure if it is derived from arbitrary or unreliable processes, even if labeled as foundational.**
