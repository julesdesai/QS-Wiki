# Can memory impact the reliability of perceptually derived knowledge?

**Node Type:** antithesis
**Node ID:** 26f3ea06-7b1f-4050-a486-4abc1cf01b43
**Depth:** 4
**Terminal:** No

**Parent:** [Direct Realism with Strong Correspondence](direct-realism-with-strong-correspondence-synthesis-28884d49-4ced-45da-83a0-528139a02f57.md)

## Content

**Memory, as an intermediary between perception and knowledge, can be faulty or unreliable, such as a misremembered event.**, **If memory distorts a perceptual experience, then the knowledge claimed from this experience may not reliably correspond to the external reality.**, **If memory errors affect perceptual knowledge, it questions the strict reliance on direct perception for reliable knowledge.**
