# How does inter-subjective phenomenalism handle perceptual illusions?

**Node Type:** antithesis
**Node ID:** af751c7b-5a38-471e-b450-65dbb3de8a2e
**Depth:** 4
**Terminal:** No

**Parent:** [Inter-subjective Phenomenalism](inter-subjective-phenomenalism-synthesis-367c4cba-19af-4964-9bd7-aba768d0b0b6.md)

## Content

**A case occurs where a group collectively perceives a mirage as water. Intuitively, this consensus does not confer true knowledge of water's existence.**, **There is a plausible principle that true knowledge must correspond to reality. If collective perception can lead to misperception, then it is implausible that collective perception alone can confer true knowledge.**
