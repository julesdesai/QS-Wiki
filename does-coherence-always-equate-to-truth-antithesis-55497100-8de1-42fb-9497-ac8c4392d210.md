# Does coherence always equate to truth?

**Node Type:** antithesis
**Node ID:** 55497100-8de1-42fb-9497-ac8c4392d210
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-71b6710d-dd35-475a-8146-f2431fcd7269.md)

## Content

**A case in which a person has a coherent set of beliefs, including one that is true, is not necessarily a case in which the person knows the true proposition.**, **There is a plausible principle that coherence alone does not grant knowledge. This principle together with the existence of coherent but false beliefs entails that coherence and truth can be independent of each other.**
