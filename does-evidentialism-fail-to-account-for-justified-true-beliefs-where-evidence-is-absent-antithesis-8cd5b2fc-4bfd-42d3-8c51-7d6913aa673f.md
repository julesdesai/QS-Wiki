# Does Evidentialism fail to account for justified true beliefs where evidence is absent?

**Node Type:** antithesis
**Node ID:** 8cd5b2fc-4bfd-42d3-8c51-7d6913aa673f
**Depth:** 4
**Terminal:** No

**Parent:** [Evidentialism Summary](evidentialism-summary-synthesis-a76d832e-447a-4168-b9be-6c867a1a7af1.md)

## Content

**Consider a scenario where a person correctly guesses the number of jellybeans in a jar without having any evidence to base their guess on. Intuitively, we would judge that the belief is justified because it is true, but evidentialism would deny this because there is no evidence.**, **It is implausible that a true belief in a case like guessing the number of jellybeans could not be justified when it turns out to be true.**
