# Is an intellectual virtue necessary if reliability can be achieved otherwise?

**Node Type:** antithesis
**Node ID:** 4f77d66c-0296-4a09-8414-bb4d55c4f527
**Depth:** 4
**Terminal:** No

**Parent:** [Expanded Virtue Reliabilism](expanded-virtue-reliabilism-synthesis-be1f6636-aeca-4898-8b76-a37fb9c31f3b.md)

## Content

**Consider a scenario in which modern technology reliably produces true beliefs for everyone interfacing with it, irrespective of their intellectual virtues. In such a scenario, intuitively, we would judge that intellectual virtue is unnecessary if reliability is guaranteed technologically.**, **There is a plausible principle that intellectual virtues are personal attributes, suggesting reliability may occur independently of these virtues.**
