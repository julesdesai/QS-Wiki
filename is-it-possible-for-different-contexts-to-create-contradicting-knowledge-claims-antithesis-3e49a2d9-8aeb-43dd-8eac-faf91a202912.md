# Is it possible for different contexts to create contradicting knowledge claims?

**Node Type:** antithesis
**Node ID:** 3e49a2d9-8aeb-43dd-8eac-faf91a202912
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism](contextualism-synthesis-b8a53bd2-d0b8-44ba-a658-8cb1f890a45a.md)

## Content

**Suppose we have a case where two people from different contexts claim knowledge that directly contradict each other. Intuitively, we either have to accept that one claim is false or that knowledge can be contradictory. If one claim is false, it undermines the contextualist perspective. If knowledge can be contradictory, it challenges the coherence of knowledge itself.**, **A plausible principle is that two contradicting statements cannot both be knowledge. This principle, when set against contextualism, reveals a potential for contradictions that could discredit knowledge claims.**
