# Is it possible for a belief to be true and justified without an appropriate causal link?

**Node Type:** antithesis
**Node ID:** 6d3c0d8e-1c68-42b2-800f-4dfe75be7bbd
**Depth:** 4
**Terminal:** No

**Parent:** [Causal Theory of Knowledge](causal-theory-of-knowledge-synthesis-a5ebdb75-c32b-47a3-8c6b-cdebc8ceca67.md)

## Content

**Consider a scenario in which a person has a justified belief that it will rain because they saw a weather report predicting rain, and it does indeed rain because of an unexpected storm.**, **In this scenario, intuitively, the belief is not caused by the truth of the proposition in a typical causal chain sense.**
