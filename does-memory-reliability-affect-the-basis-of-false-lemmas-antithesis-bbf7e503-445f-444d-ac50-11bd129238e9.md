# Does memory reliability affect the basis of false lemmas?

**Node Type:** antithesis
**Node ID:** bbf7e503-445f-444d-ac50-11bd129238e9
**Depth:** 4
**Terminal:** No

**Parent:** [No False Lemmas View](no-false-lemmas-view-synthesis-031e6f6f-2b49-47e4-821c-d2421562acef.md)

## Content

**Consider a case in which a person remembers a past event inaccurately but believes their memory is accurate. Intuitively, we either have to say that their belief about the event is justified or that it is not. If we say their belief is justified, then unreliable memory can lead to false lemmas. If we say it is not justified, then memory reliability is crucial for avoiding false lemmas.**
