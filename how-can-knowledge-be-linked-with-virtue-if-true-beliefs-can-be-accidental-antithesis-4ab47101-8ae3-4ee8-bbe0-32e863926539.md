# How can knowledge be linked with virtue if true beliefs can be accidental?

**Node Type:** antithesis
**Node ID:** 4ab47101-8ae3-4ee8-bbe0-32e863926539
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-39ebccb5-104c-44e1-841f-55d82b6e37fc.md)

## Content

**There is a plausible principle that genuine knowledge cannot be based on luck. This principle together with the idea of a knower's intellectual virtues leading to true beliefs suggests that true beliefs arising accidentally do not constitute knowledge.**, **Consider a scenario in which a person forms a true belief that it will rain tomorrow because they have a reliable feeling about the weather. However, unbeknownst to them, the weather prediction was actually based on a faulty report. Intuitively, such a belief seems more lucky than virtuous, questioning whether it truly qualifies as knowledge.**
