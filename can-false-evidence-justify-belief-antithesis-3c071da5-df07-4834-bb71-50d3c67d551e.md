# Can false evidence justify belief?

**Node Type:** antithesis
**Node ID:** 3c071da5-df07-4834-bb71-50d3c67d551e
**Depth:** 4
**Terminal:** No

**Parent:** [Evidentialism](evidentialism-synthesis-bfc3ee3a-740c-4ee5-bb7b-12c5efc1c03c.md)

## Content

**Suppose we have a case in which a person believes that it is raining because they see someone carrying an umbrella. Intuitively, we would say that if it turns out the person is carrying an umbrella for sun protection, the evidence does not justify the belief that it is raining.**, **There is a plausible principle that false evidence cannot justify a belief in its truth. This principle, together with the intuition about the scenario, suggests that beliefs supported by false evidence are not justified.**
