# Do all knowledge claims require true beliefs from properly functioning faculties?

**Node Type:** antithesis
**Node ID:** 8da5fe30-b794-4be6-bcf8-b61b395f3d4b
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-7d89ec66-0836-44e3-8613-d999cc659dfb.md)

## Content

**Suppose there are cases where someone forms a true belief by chance rather than through properly functioning cognitive faculties aiming at truth. Intuitively, we either have to say that such a belief is still knowledge or that it is not. If we say it is, then the Proper Function Theory is challenged.**
