# Does mere belief suffice for knowledge?

**Node Type:** antithesis
**Node ID:** 688ce341-0340-4cee-94e0-0ba135869eaa
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Epistemology](contextualist-epistemology-synthesis-771a8ab1-0860-48e1-822c-9b3b2cda1c63.md)

## Content

**Consider a scenario in which a person believes a proposition due to random guesswork, and it coincidentally turns out to be true. Intuitively, this belief does not constitute genuine knowledge.**, **There is a plausible principle that knowledge requires more than mere true belief; it also requires some form of justification.**, **A case in which a person believes that a coin will land heads, and it does so purely by chance, is not a case in which the person knows it will land heads.**
