# Does perception without the corresponding external objects challenge intentionalism?

**Parent:** [Intentionalism Summary](intentionalism-summary-synthesis-3678fe22-b4a0-4fa7-aad8-5dfa98b1a709.md)

## Content

**Consider a scenario where a person hallucinates a pink elephant. Perception has intentional content that represents external reality would require this hallucination to have a real-world counterpart.**, **It is implausible that a non-existent object like a pink elephant corresponds to external reality.**
