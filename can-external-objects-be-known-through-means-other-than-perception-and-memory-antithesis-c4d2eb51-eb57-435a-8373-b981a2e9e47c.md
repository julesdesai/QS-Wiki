# Can external objects be known through means other than perception and memory?

**Node Type:** antithesis
**Node ID:** c4d2eb51-eb57-435a-8373-b981a2e9e47c
**Depth:** 4
**Terminal:** No

**Parent:** [Memory-Integrated Phenomenalism](memory-integrated-phenomenalism-synthesis-20b6ad46-c9b1-41b8-87a0-7619b03b72c5.md)

## Content

**Suppose a scenario where a person learns about an external object through logical deduction based on premises that are known a priori. Intuitively, knowing an external object this way does not depend on perceptual experiences or memory integration.**, **It is implausible that all knowledge about external objects must stem from perception if deduction alone can provide knowledge of external objects.**
