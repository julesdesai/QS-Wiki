# Does coherence account for the diversity of perceptual experiences?

**Node Type:** antithesis
**Node ID:** 65fea87b-362b-464f-b9d5-91c3f1426bad
**Depth:** 4
**Terminal:** No

**Parent:** [Phenomenalism with Constructive Coherence](phenomenalism-with-constructive-coherence-synthesis-649780d2-d21b-4c5f-9b9f-162351fea396.md)

## Content

**Suppose we have a case in which multiple people perceive the same event differently, with each account consistent within itself. Intuitively, we either have to say that all divergent perceptions correspond to different truths or that only one is true and the rest are illusory. If we say all are true, then coherence cannot maintain a single reality. If we say only one is true, coherence alone doesn't help identify truth.**, **There is a plausible principle that coherence should help distinguish true perceptions from false ones. This principle together with we observe diversity in perceptions breeds inconsistency, challenging the utility of coherence.**
