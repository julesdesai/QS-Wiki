# Is it possible for a belief to be true without being causally grounded in the truth?

**Node Type:** antithesis
**Node ID:** 1337801c-7920-42ff-8c87-5abc6e20e7d7
**Depth:** 4
**Terminal:** No

**Parent:** [Causal Theory of Knowledge](causal-theory-of-knowledge-synthesis-916d423a-1723-4911-b4ab-5dfda1d6916a.md)

## Content

**There is a plausible principle that believes must be causally grounded in their truth to be considered knowledge. If a student guesses the answer on a test and it happens to be true unrelated to any causal justification, one would intuitively not regard this as knowledge, suggesting the causal theory's limitations.**
