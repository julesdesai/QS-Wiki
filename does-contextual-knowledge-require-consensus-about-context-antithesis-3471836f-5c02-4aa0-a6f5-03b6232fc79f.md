# Does contextual knowledge require consensus about context?

**Node Type:** antithesis
**Node ID:** 3471836f-5c02-4aa0-a6f5-03b6232fc79f
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Foundationalism](contextualist-foundationalism-synthesis-b0409110-d832-4fdb-9fe3-2e01026d1f88.md)

## Content

**Suppose we have a case in which two equally rational observers disagree about the contextual relevance of a foundational belief. Intuitively, either both possess knowledge or neither does. If we say both have knowledge, then knowledge depends only on individual interpretation. If we say neither has knowledge, then consensus about context seems necessary.**
