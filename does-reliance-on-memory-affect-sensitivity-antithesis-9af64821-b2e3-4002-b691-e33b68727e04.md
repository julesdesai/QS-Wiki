# Does reliance on memory affect sensitivity?

**Node Type:** antithesis
**Node ID:** 9af64821-b2e3-4002-b691-e33b68727e04
**Depth:** 4
**Terminal:** No

**Parent:** [Sensitivity Theory](sensitivity-theory-synthesis-68f75ad1-a4e1-40d8-ae80-33e70d924463.md)

## Content

**Consider a scenario where a person remembers a true proposition from a past lecture. In relevant situations where the proposition is false, the person’s reliance on memory causes them to continuously believe it. Intuitively, this over-reliance on memory without re-evaluation seems to lack genuine sensitivity.**
