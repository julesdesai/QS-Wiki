# What if cognitive faculties are reliable but unknowingly manipulated?

**Node Type:** antithesis
**Node ID:** 86e24f78-54b4-4e01-a1d9-97d59370937a
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Functionalist Knowledge Summary](proper-functionalist-knowledge-summary-synthesis-ea5a08e3-f799-4ced-bdad-582e7b9aaf66.md)

## Content

**Consider a scenario where an individual has cognitive faculties manipulated by an external agent to generate only true beliefs. Intuitively, this manipulation undermines the idea of genuine knowledge.**, **It's plausible that knowledge requires independent functioning of cognitive faculties.**
