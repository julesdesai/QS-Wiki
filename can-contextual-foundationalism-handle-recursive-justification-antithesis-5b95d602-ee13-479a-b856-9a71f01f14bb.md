# Can contextual foundationalism handle recursive justification?

## Content

**A case in which a foundational belief itself requires further foundational beliefs for justification challenges the idea of self-contained contextually justified foundations.**, **Intuitively, if each foundational belief requires justification, the structure may become recursive, lacking a solid stopping point.**
