# Can you know something if the belief is true but held for the wrong reasons?

**Node Type:** antithesis
**Node ID:** ca8a3393-5f25-4405-97ca-f883a05d3b17
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology](virtue-epistemology-synthesis-b6d42ffa-59fb-457e-877f-3bcb8ac1af86.md)

## Content

**Consider a case where a person holds a true belief based purely on superstition, like believing it will rain tomorrow because they see a black cat.**, **A belief based on superstition does not stem from an intellectual virtue like evidence-based reasoning.**, **Holding a belief for the wrong reasons does not align with the exercise of cognitive virtue.**
