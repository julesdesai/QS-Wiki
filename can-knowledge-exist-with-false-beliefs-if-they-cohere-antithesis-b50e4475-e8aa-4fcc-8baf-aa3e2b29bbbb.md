# Can knowledge exist with false beliefs if they cohere?

**Node Type:** antithesis
**Node ID:** b50e4475-e8aa-4fcc-8baf-aa3e2b29bbbb
**Depth:** 2
**Terminal:** No

**Parent:** [Coherentism](coherentism-thesis-368b08e4-1a7c-4e5d-96a8-85ade01bb443.md)

## Content

**It is possible for a set of beliefs to be internally coherent and yet contain false beliefs.**, **A belief system in which all its elements support each other does not guarantee the truth of any particular belief within that system.**
