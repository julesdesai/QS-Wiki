# Can a belief system that coheres yet lacks core beliefs anchored in reality still be considered knowledge?

**Node Type:** antithesis
**Node ID:** f8982fa4-7a44-4e1a-b834-99df4db7921f
**Depth:** 4
**Terminal:** No

**Parent:** [Partial Correspondence Coherentism Summary](partial-correspondence-coherentism-summary-synthesis-73282c9a-d730-43f4-a933-7c2ffa7c9bab.md)

## Content

**A belief system could be entirely coherent internally even if none of the core beliefs correspond to actual states of affairs.**, **It is plausible that a belief system need not correspond to reality in order to be considered coherent.**, **If coherence alone without correspondence is sufficient, then the requirement that at least some parts of the belief system are anchored in reality is unnecessary for knowledge.**
