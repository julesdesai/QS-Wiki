# Can Contextualism handle disagreements across contexts?

**Node Type:** antithesis
**Node ID:** ea202a4e-684e-4d73-b961-c668b01fec8b
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist View](contextualist-view-synthesis-ee61dbb1-ab63-492e-a056-b4bb19752825.md)

## Content

**Consider a case where two individuals with different contextual standards disagree on whether a belief counts as knowledge.**, **There is a plausible principle that genuine disagreement requires a common standard.**, **Intuitively, if there is no common standard due to contextual variations, it undermines the possibility of genuine disagreement about knowledge.**
