# Can we have justified false beliefs in a coherent system?

**Node Type:** antithesis
**Node ID:** a7a152bd-6018-4841-9455-9e21856fc4fc
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentist Knowledge Summary](coherentist-knowledge-summary-synthesis-1fa3c944-9677-491a-8edd-ea154ce998c7.md)

## Content

**Consider a scenario in which all beliefs in a system are coherent but based on a false premise. In such a scenario, intuitively we would judge that the belief is justified but not knowledge.**, **It is plausible that coherence allows for justified false beliefs if the system is internally consistent. Justification in such a case does not lead to knowledge.**
