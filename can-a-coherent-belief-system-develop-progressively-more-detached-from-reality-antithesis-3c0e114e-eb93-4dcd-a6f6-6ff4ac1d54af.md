# Can a coherent belief system develop progressively more detached from reality?

**Node Type:** antithesis
**Node ID:** 3c0e114e-eb93-4dcd-a6f6-6ff4ac1d54af
**Depth:** 4
**Terminal:** No

**Parent:** [Coherence Theory of Knowledge](coherence-theory-of-knowledge-synthesis-3d6de9a5-e71e-464f-88a2-666531bfab5e.md)

## Content

**A case in which an individual starts with a coherent but incorrect assumption about the workings of the economy and builds an increasingly elaborate and consistent belief system. This case is not one in which the coherence ensures eventual convergence with truth.**, **There is a plausible principle that without external correction, coherent belief systems can become increasingly entrenched in false presuppositions. This principle together with coherence theory suggests that coherence does not inherently correct false beginnings.**
