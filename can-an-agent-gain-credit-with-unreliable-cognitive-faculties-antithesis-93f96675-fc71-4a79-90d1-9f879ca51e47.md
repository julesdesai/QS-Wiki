# Can an agent gain credit with unreliable cognitive faculties?

**Node Type:** antithesis
**Node ID:** 93f96675-fc71-4a79-90d1-9f879ca51e47
**Depth:** 4
**Terminal:** No

**Parent:** [Credit Theory of Knowledge](credit-theory-of-knowledge-synthesis-f65ebe79-7fd3-4c3c-9fb3-cd47f9656422.md)

## Content

**Consider a scenario in which a person forms a true belief about the time based on a clock that is stopped but happens to display the correct time. In such a scenario, intuitively we would judge that the person does not deserve credit for having a true belief.**, **The agent does not employ intellectual virtues, but rather luck, in obtaining the true belief.**
