# Can purely logical inferences ensure absence of false lemmas?

**Node Type:** antithesis
**Node ID:** ae581c61-cd83-48e2-93b3-b036350f041e
**Depth:** 4
**Terminal:** No

**Parent:** [No False Lemmas View](no-false-lemmas-view-synthesis-031e6f6f-2b49-47e4-821c-d2421562acef.md)

## Content

**Consider a scenario where a person uses logical reasoning to infer a true conclusion from a set of premises, but one of these premises is false although the conclusion is accidentally correct. In such a scenario, intuitively we would judge that the reasoning does not yield knowledge.**, **There is a plausible principle that logical reasoning should not rely on false premises to produce knowledge. This principle together with the person's accurate conclusion from a false premise entails that accurate logical inferences may not ensure knowledge if any lemma is false.**
