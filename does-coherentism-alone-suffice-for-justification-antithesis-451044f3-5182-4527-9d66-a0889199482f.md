# Does coherentism alone suffice for justification?

**Node Type:** antithesis
**Node ID:** 451044f3-5182-4527-9d66-a0889199482f
**Depth:** 4
**Terminal:** No

**Parent:** [Foundationalist Coherentism](foundationalist-coherentism-synthesis-1575945f-cea5-4275-8c31-c7cabab6c71a.md)

## Content

**There is a plausible principle that if all beliefs within a system support each other coherently, the system is justified. This calls into question the necessity of a foundational belief.**, **Consider a scenario where all beliefs are coherently interconnected but lack a foundational belief. Intuitively, the system still seems justified.**
