# Is it possible to have false knowledge?

**Node Type:** antithesis
**Node ID:** d7d82c53-3122-4af9-a510-3b3a787cc980
**Depth:** 4
**Terminal:** No

**Parent:** [Refined Virtue Epistemology](refined-virtue-epistemology-synthesis-ed8e6ab8-d432-4955-9aec-982a4b148be7.md)

## Content

**Truth is a necessary component for knowledge. This requirement implies that any proposition believed that turns out to be false cannot count as knowledge.**, **There is a plausible principle that claims something false can never be known, which implies that knowledge must always be true.**
