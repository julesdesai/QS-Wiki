# Can a true belief arrived at by unreliable virtue be considered knowledge?

**Node Type:** antithesis
**Node ID:** 77622b68-dde0-4f16-8d6f-e09af6b88c84
**Depth:** 4
**Terminal:** No

**Parent:** [Enhanced Virtue Reliabilism with Error Avoidance](enhanced-virtue-reliabilism-with-error-avoidance-synthesis-bae843f7-47b1-4791-af4f-a08b903c892d.md)

## Content

**Suppose we have a scenario where someone forms a true belief using a virtue that occasionally produces error, but in this case, it works correctly. Intuitively, we would judge that this should not count fully as knowledge.**, **There is a plausible principle that occasional success due to imperfect virtues does not suffice for reliable error avoidance required for knowledge, implying fallibility undermines knowledge.**
