# Does hypothetical error affecting belief source affect knowledge?

**Node ID:** 0400368f-48e8-42c3-a895-b6c1aaad98b3
**Depth:** 4
**Terminal:** No

**Parent:** [No-False-Lemma Theory](no-false-lemma-theory-synthesis-4a11ee42-2be9-4836-9dd6-d365881bb277.md)

## Content

**Consider a scenario in which a person learns about a historical event from a reputable source, but a historical error elsewhere in the text exists. Intuitively, this hypothetical error challenges the full certainty of knowledge of the event learned.**, **Suppose an ancient text often cited by historians mentions a fact. If it emerges the text has some misrepresented dates elsewhere, then even correctly presented facts create doubt as a reference for knowledge.**
