# Is the set of foundational beliefs truly minimal?

**Node Type:** antithesis
**Node ID:** 8b7b1f81-a646-4fa1-bac9-a41a46642f55
**Depth:** 4
**Terminal:** No

**Parent:** [Minimal Foundational Coherentism Summary](minimal-foundational-coherentism-summary-synthesis-0e7aa745-4942-4fbb-a96c-6043bbe35e5d.md)

## Content

**Suppose we have a belief system that necessitates a large number of empirical beliefs to avoid isolation. Intuitively, this contradicts the idea that there is a "minimal" set of beliefs.**, **The existence of such empirical beliefs challenges the notion of a system that relies on minimal foundational beliefs, as expanding the foundational base suggests non-minimality.**
