# How does invariantism handle disagreement between knowledgeable parties?

**Node Type:** antithesis
**Node ID:** da686ade-568b-414e-86f6-b0581690d6d3
**Depth:** 4
**Terminal:** No

**Parent:** [Invariantism with Fixed Standards](invariantism-with-fixed-standards-synthesis-2877eecc-8388-4e01-b87d-ffe7f20b2619.md)

## Content

**Two knowledgeable experts in different fields might both claim certainty about an overlapping subject, yet intuitively both cannot know if one is mistaken.**, **If knowledge is independent of context, both experts should have knowledge, which seems impossible if their claims conflict.**, **Invariantism must account for knowledge being affected by the differing epistemic standards between conflicting experts, but it does not.**
