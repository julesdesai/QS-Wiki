# Do context-sensitive beliefs withstand cross-context scrutiny?

**Node Type:** antithesis
**Node ID:** 7e6bbc19-dfd0-4c11-b5ca-be5344d5074c
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Foundationalism](contextualist-foundationalism-synthesis-b757e054-5c50-4073-a4ee-c1ee757cbdbb.md)

## Content

**Suppose the belief that a plant is healthy is justified because it looks green and upright; however, light conditions reveal poor health. Another context advises thorough examination beyond visual cues.**, **A belief is justified if it is appropriate to the context in which it is assessed; however, this fails when contexts impose different evaluative standards.**, **It is implausible that context-specific justification prevents error when contexts demand different criteria for validation.**
