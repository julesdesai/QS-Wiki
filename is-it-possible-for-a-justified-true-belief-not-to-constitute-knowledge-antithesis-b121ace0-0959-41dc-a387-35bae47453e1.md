# Is it possible for a justified true belief not to constitute knowledge?

**Node Type:** antithesis
**Node ID:** b121ace0-0959-41dc-a387-35bae47453e1
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism with a Basic Belief Component](coherentism-with-a-basic-belief-component-synthesis-59782134-c469-4431-baca-a56ff4989242.md)

## Content

**A scenario in which someone has a justified true belief because they have guessed and been lucky is a case where the person does not have knowledge**, **A case where someone guesses correctly and has a belief that is both true and justified intuitively is not a case of knowledge**, **In many intuitive cases, having a justified true belief does not suffice for knowledge**
