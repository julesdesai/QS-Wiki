# What if non-cognitive mechanisms produce true beliefs as knowledge?

**Depth:** 4
**Terminal:** No

## Content

**Imagine a scenario in which a person receives true beliefs through telepathy from a truth-possessed source, without any cognitive mechanism being involved. Intuitively, we might still describe this as knowledge.**, **True beliefs need not arise from cognitive mechanisms alone to be considered knowledge.**
