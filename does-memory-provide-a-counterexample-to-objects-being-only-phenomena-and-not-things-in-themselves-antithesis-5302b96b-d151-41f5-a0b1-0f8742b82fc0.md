# Does memory provide a counterexample to objects being only phenomena and not things in themselves?

**Node ID:** 5302b96b-d151-41f5-a0b1-0f8742b82fc0

## Content

**Consider an accurate memory of a past event without current sensory input suggesting the event's independent existence.**, **If memory yields knowledge without immediate perception, it supports belief in things existing as they are, beyond experiential phenomena.**, **Memory's reliability in reconstructing past truths independently of current phenomena questions the limitation to knowing only phenomena.**
