# Is it possible for a belief to be true and safely held yet not be true in the actual world?

**Node Type:** antithesis
**Node ID:** 330d4549-f6ac-444d-b4da-748206d7d9f9
**Depth:** 4
**Terminal:** No

**Parent:** [Knowledge as Safety](knowledge-as-safety-synthesis-df26c783-c115-4102-a084-91df0b928704.md)

## Content

**There is a plausible principle that the truth of a belief must be evaluated in the actual world for it to count as knowledge.**, **Suppose we have a belief that is true and safely held in nearby possible worlds, but in the actual world, the proposition is not true, intuitively we judge that the person does not have knowledge.**
