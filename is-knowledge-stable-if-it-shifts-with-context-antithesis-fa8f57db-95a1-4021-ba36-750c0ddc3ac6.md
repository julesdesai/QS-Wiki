# Is knowledge stable if it shifts with context?

**Node Type:** antithesis
**Node ID:** fa8f57db-95a1-4021-ba36-750c0ddc3ac6
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Theory of Knowledge](contextualist-theory-of-knowledge-synthesis-4769a94f-2567-4d65-9377-c3f1aea872f2.md)

## Content
