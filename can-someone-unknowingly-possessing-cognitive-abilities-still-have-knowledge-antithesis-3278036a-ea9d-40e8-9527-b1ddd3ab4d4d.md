# Can someone unknowingly possessing cognitive abilities still have knowledge?

**Node Type:** antithesis
**Node ID:** 3278036a-ea9d-40e8-9527-b1ddd3ab4d4d
**Depth:** 4
**Terminal:** No

**Parent:** [Credit Theory of Knowledge](credit-theory-of-knowledge-synthesis-193f50ad-c71a-4e26-bf9b-21d5bbf78043.md)

## Content

**Imagine a person P with an exceptional memory who routinely forms true beliefs but does not know about this ability. If P forms a true belief about an event based on memory alone, intuitively, we might say they have knowledge despite not recognizing their own cognitive skills.**, **It is implausible that cognitive credit requires awareness of one's abilities since unrecognized abilities can still lead to knowledge.**
