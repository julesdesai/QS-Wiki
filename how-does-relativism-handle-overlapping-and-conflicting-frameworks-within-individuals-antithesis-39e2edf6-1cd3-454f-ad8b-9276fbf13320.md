# How does relativism handle overlapping and conflicting frameworks within individuals?

**Node Type:** antithesis
**Node ID:** 39e2edf6-1cd3-454f-ad8b-9276fbf13320
**Depth:** 4
**Terminal:** No

**Parent:** [Relativism Summary](relativism-summary-synthesis-2665e075-79d4-4adb-9371-55cb5a5794f8.md)

## Content

**Suppose we have a case in which a person belongs to multiple cultural frameworks with conflicting beliefs about a single proposition. Intuitively, this raises the issue of which framework should prevail.**, **It is implausible that one can consistently hold contradictory beliefs as true based on multiple frameworks simultaneously.**
