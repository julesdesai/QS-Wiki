# Does reliance on testimony introduce false lemmas?

**Node Type:** antithesis
**Node ID:** 507c7530-c898-4d8b-8734-6694d608282a
**Depth:** 4
**Terminal:** No

**Parent:** [No False Lemmas View](no-false-lemmas-view-synthesis-031e6f6f-2b49-47e4-821c-d2421562acef.md)

## Content

**Consider a case in which a person forms a belief based on testimony from an unreliable witness who happens to report the truth this time. Intuitively, we either have to say that the belief counts as knowledge or that it does not. If we say it counts as knowledge, then the principle that unreliable sources cannot justify knowledge is violated. If we say it does not count as knowledge, then reliance on potentially false lemmas is inherent in testimonial justification.**
