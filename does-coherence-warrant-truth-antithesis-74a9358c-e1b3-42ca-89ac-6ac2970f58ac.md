# Does coherence warrant truth?

**Node Type:** antithesis
**Node ID:** 74a9358c-e1b3-42ca-89ac-6ac2970f58ac
**Depth:** 4
**Terminal:** No

**Parent:** [Coherent Phenomenalism](coherent-phenomenalism-synthesis-2365a190-e3fc-4d37-a83b-defd0eadffa2.md)

## Content

**Consider a scenario in which two people have equally coherent but mutually exclusive sets of perceptual experiences regarding the same event. Intuitively, both cannot possess true knowledge because the event cannot occur in both mutually exclusive ways.**, **There is a plausible principle that if two coherent beliefs contradict each other, it is not possible for both to be true simultaneously, thus coherence does not guarantee truth.**
