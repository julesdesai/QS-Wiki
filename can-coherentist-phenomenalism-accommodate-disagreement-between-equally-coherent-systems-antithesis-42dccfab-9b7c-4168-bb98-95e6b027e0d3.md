# Can coherentist phenomenalism accommodate disagreement between equally coherent systems?

**Node Type:** antithesis
**Node ID:** 42dccfab-9b7c-4168-bb98-95e6b027e0d3
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentist Phenomenalism](coherentist-phenomenalism-synthesis-7f881170-8b02-4aad-92a8-2bed5a4b8a3c.md)

## Content

**It is possible for two individuals to have entirely coherent but mutually contradictory belief systems about the same situation.**, **Each system, while internally coherent, cannot both be true if they conflict over the same objective facts.**, **If coherentist phenomenalism allows contradictory yet coherent belief systems to count as knowledge, it may struggle to resolve such disagreements.**
