# How can context-sensitivity address conflicting intuitions about the same knowledge claim?

**Node Type:** antithesis
**Node ID:** ec5489ca-8a4e-4a45-b38a-9cfef01674e7
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism](contextualism-synthesis-1e9d0e24-887b-4fad-bf5c-71b82495d573.md)

## Content

**Consider a scenario in which a person sincerely believes they know a proposition while in one context but doubts their knowledge when the context changes without any new evidence. Intuitively, it seems implausible that knowledge can fluctuate in this manner.**, **There is a plausible principle that knowledge should remain stable if there is no change in evidence or justification. This principle together with knowledge's fluctuation due to context change entails an inconsistency.**
