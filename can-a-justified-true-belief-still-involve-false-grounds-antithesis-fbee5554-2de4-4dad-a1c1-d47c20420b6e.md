# Can a justified true belief still involve false grounds?

**Node Type:** antithesis
**Node ID:** fbee5554-2de4-4dad-a1c1-d47c20420b6e
**Depth:** 4
**Terminal:** No

**Parent:** [No False Grounds Theory](no-false-grounds-theory-synthesis-37dfac32-afee-4ea8-9a67-23e7efbc2507.md)

## Content

**There is a plausible principle that justified true belief can still fail to constitute knowledge if the belief inadvertently relies on false grounds.**, **Suppose we have a case where the justification for a true belief is based on a mistaken but seemingly reliable premise. Intuitively, this might suggest that the belief is not true knowledge due to its reliance on error.**
