# Can contextualism account for shared knowledge across different contexts?

**Node Type:** antithesis
**Node ID:** 75b6a8f8-128e-4741-a2bc-2dfefdbda2a6
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism](contextualism-synthesis-605b5e27-4923-4bc2-b3cf-30db710f9ba8.md)

## Content

**A belief is justified if it meets the standards set by the context.**, **Shared knowledge presupposes common standards for justification accepted in multiple contexts.**, **If standards are entirely context-dependent, shared knowledge could be undermined unless commonality between contexts is established.**
