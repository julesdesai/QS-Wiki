# Can cognitive faculties determine truth without faulty design?

**Node Type:** antithesis
**Node ID:** 7431cd0a-6c02-4ecf-8e4b-cf1b26dec3b9
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-ea617099-7596-484d-aca6-1b318b0033ac.md)

## Content

**Consider the case where a person's cognitive faculties are biologically malfunctioning but still generate true beliefs. Intuitively, it seems that the person does not have knowledge.**, **A person can have a true belief based on cognitive faculties that are not functioning according to any coherent design plan, which suggests that proper function might not be necessary for knowledge.**
