# Are intellectual virtues necessary if memory alone supports skill?

**Node Type:** antithesis
**Node ID:** b7c0da68-7c69-4f82-adc3-424ec3b5ccab
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology with Skills](virtue-epistemology-with-skills-synthesis-458da717-0537-4f5a-8a52-aec5d63f6438.md)

## Content

**Suppose we have a case where a person performs a musical piece flawlessly from memory with no active intellectual virtue engaged. Intuitively, we either have to say that memory-driven performance is sufficient for knowledge or that a deeper cognitive virtue is necessary. If we say memory is enough, it challenges the necessity of intellectual virtues for knowledge. If we say intellectual virtues are necessary, it questions skill-based performance as knowledge.**
