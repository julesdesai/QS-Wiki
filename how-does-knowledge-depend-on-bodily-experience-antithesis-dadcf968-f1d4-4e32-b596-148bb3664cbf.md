# How does knowledge depend on bodily experience?

**Node Type:** antithesis
**Node ID:** dadcf968-f1d4-4e32-b596-148bb3664cbf
**Depth:** 4
**Terminal:** No

**Parent:** [Embodied Cognition with Reliance on Bodily Skills](embodied-cognition-with-reliance-on-bodily-skills-synthesis-e9d28d24-22c1-4ccc-a9a1-a716398534d0.md)

## Content

**Suppose we have a case in which a computer system gains true beliefs through data processing, without any bodily engagement. Intuitively, we either have to say that this system possesses knowledge or that it does not. If we say it possesses knowledge, then bodily experience is not necessary. If we say it does not, then something besides true belief is essential for knowledge.**
