# Can knowledge arise from an unreliable cognitive process?

**Node Type:** antithesis
**Node ID:** c9958569-0b1b-4281-af72-7f68ca8a9bf3
**Depth:** 4
**Terminal:** No

**Parent:** [Reliability Theory of Knowledge](reliability-theory-of-knowledge-synthesis-74a39329-93a9-479f-b8d3-ff182da0738f.md)

## Content

**Suppose we have a case in which a person believes a proposition is true because a notoriously unreliable horoscope suggests it. Intuitively, this belief would not constitute knowledge.**, **The belief is produced by a process that does not typically yield true beliefs, yet ends up being true. This provides true belief without reliable knowledge.**, **The horoscope happens to be correct by sheer luck, but luck undermines the reliability requirement for knowledge.**, **This illustrates that true belief from an unreliable process can sometimes be correct, suggesting reliability may not be necessary for knowledge.**
