# Is reliability necessary for knowledge if it allows for strange scenarios?

**Node Type:** antithesis
**Node ID:** b1d32fa4-8875-440a-9c17-f2c10a222594
**Depth:** 4
**Terminal:** No

**Parent:** [Reliabilism](reliabilism-synthesis-a142f10a-5d37-4074-9f91-84a5b31f1b2a.md)

## Content

**Consider a scenario where a person forms a true belief by rolling a die and the result is consistently accurate despite the randomness. Intuitively, one might judge this person does not genuinely possess knowledge.**, **There is a plausible principle that knowledge should exclude methods dependent on chance or luck. This principle together with the notion that mere reliability suffices for knowledge suggests a conflict when randomness results in reliable outcomes.**
