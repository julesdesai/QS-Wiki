# Does contextualism allow for skepticism in high-stakes cases?

**Node Type:** antithesis
**Node ID:** 4622a0f1-89eb-499f-aaa0-502319abb98a
**Depth:** 4
**Terminal:** No

**Parent:** [Moderate Contextualism](moderate-contextualism-synthesis-4577cf68-8dec-49df-bf7d-5b1d9d39d213.md)

## Content

**Consider a case in which an individual doubts a well-supported proposition because the high-stakes context raises the knowledge standards to an impractically high level. Intuitively, this allows for skepticism where there should be certainty.**, **There is a plausible principle that knowledge should offer reassurance or closure about what is known. This principle together with raised standards in high-stakes contexts suggests that skepticism might be induced when it is unwarranted.**
