# How does Contextual Foundationalism handle Gettier cases?

**Node Type:** antithesis

**Depth:** 4
**Terminal:** No

**Parent:** [Contextual Foundationalism](contextual-foundationalism-synthesis-3cd555af-8ec3-406f-b70a-64f89852bc49.md)

## Content

**Consider a scenario where a person has a justified true belief due to a lucky coincidence, such as believing it will rain because of a misleading weather report and it rains by chance. In this scenario, intuitively we would judge that the person does not really know it will rain.**, **There is a plausible principle that for a belief to count as knowledge, its truth should not merely be a result of luck or coincidence. This principle suggests that justified true belief as outlined might not be sufficient for knowledge.**
