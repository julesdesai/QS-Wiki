# Can a single belief entail reliability?

## Content

**It is implausible that a single belief instance can adequately measure the reliability of a cognitive process.**, **Consider a scenario in which a person forms a true belief by sheer luck through a rarely reliable process. Intuitively, we would judge that reliability requires more than one instance.**
