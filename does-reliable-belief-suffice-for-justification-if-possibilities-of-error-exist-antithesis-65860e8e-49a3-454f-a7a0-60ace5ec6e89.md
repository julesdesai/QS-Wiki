# Does reliable belief suffice for justification if possibilities of error exist?

**Node Type:** antithesis
**Node ID:** 65860e8e-49a3-454f-a7a0-60ace5ec6e89
**Depth:** 4
**Terminal:** No

**Parent:** [Fallibilist Foundationalism](fallibilist-foundationalism-synthesis-ed3484e7-a0b3-47a4-939c-ec83c0f7bb7c.md)

## Content

**A belief is justified if it rests on foundational beliefs that are fallible but generally reliable.**, **Suppose we have a case where a person's belief about the weather is based on a reliable weather app that occasionally makes errors. Intuitively, we have to say that either the belief is justified despite the errors or it is not truly justified. If we say the belief is justified despite the possibility of error, then we accept that justification does not require elimination of error possibilities. If we say the belief is not justified due to error possibilities, then mere reliability does not suffice for justification.**
