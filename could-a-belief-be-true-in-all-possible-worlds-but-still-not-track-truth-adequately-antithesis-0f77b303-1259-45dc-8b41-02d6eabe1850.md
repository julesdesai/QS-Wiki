# Could a belief be true in all possible worlds but still not track truth adequately?

**Node Type:** antithesis
**Node ID:** 0f77b303-1259-45dc-8b41-02d6eabe1850
**Depth:** 4
**Terminal:** No

**Parent:** [Truth-Tracking View](truth-tracking-view-synthesis-ffb314b5-26cd-4e37-96a0-e81b3c75b895.md)

## Content

**Suppose there is a case where a belief is necessarily true in all possible worlds, such as a tautology. Intuitively, one might think this belief does not track truth in the sense required for knowledge because it is true irrespective of the facts.**, **It is implausible to treat necessarily true beliefs as tracking truth since their truth does not depend on alignment with changing scenarios or facts.**
