# Can a coherent belief system evade all external criticism?

**Node Type:** antithesis
**Node ID:** be411f72-faf3-4ff2-b638-9efe6c4394fe
**Depth:** 4
**Terminal:** No

**Parent:** [Adaptive Coherentism](adaptive-coherentism-synthesis-f17f0588-6de8-4c0d-a6e6-63e09dfee45f.md)

## Content

**Suppose we have a case in which a community has an internally coherent set of beliefs disconnected from external reality. Intuitively, we either have to say that their beliefs are justified or not. If we say they are justified, then coherence appears to evade reality checks. If we say they are not justified, then coherence alone is not sufficient.**
