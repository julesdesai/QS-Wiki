# Does reliance on memory in belief formation challenge externalism’s emphasis on reliability?

**Node Type:** antithesis
**Node ID:** cba76a83-4457-439a-ab73-5cfa3c3286d0
**Depth:** 4
**Terminal:** No

**Parent:** [Externalism With Reliable Belief Formation](externalism-with-reliable-belief-formation-synthesis-2a34c494-e349-4b40-95af-530f8f0219c4.md)

## Content

**Suppose a person forms a belief based on a memory which was initially formed by a reliable process, but the memory is now distorted. Intuitively, such distortion would undermine the knowledge claim of the current belief.**, **It is implausible that knowledge can be attributed if the reliability of the belief-forming process has been compromised over time or recollection.**
