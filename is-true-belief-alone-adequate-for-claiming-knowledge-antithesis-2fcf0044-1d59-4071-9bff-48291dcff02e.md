# Is true belief alone adequate for claiming knowledge?

**Node Type:** antithesis
**Node ID:** 2fcf0044-1d59-4071-9bff-48291dcff02e
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualist Epistemology](contextualist-epistemology-synthesis-771a8ab1-0860-48e1-822c-9b3b2cda1c63.md)

## Content

**There is a plausible principle that for a belief to count as knowledge, the truth of the proposition must be non-accidental for the believer.**, **Suppose we have a case in which a person believes a true proposition based on a defective source, like a known unreliable newspaper. Intuitively, this is not a case of knowledge because the truth seems accidental.**, **It is implausible that merely believing a true proposition without adequate justification qualifies as knowledge.**
