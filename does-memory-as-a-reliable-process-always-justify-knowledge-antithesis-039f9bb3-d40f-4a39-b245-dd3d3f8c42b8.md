# Does memory as a reliable process always justify knowledge?

**Node Type:** antithesis
**Node ID:** 039f9bb3-d40f-4a39-b245-dd3d3f8c42b8
**Depth:** 4
**Terminal:** No

**Parent:** [Externalist View](externalist-view-synthesis-75028f40-4ef7-4feb-a21d-1e6a75d787a6.md)

## Content

**Memory can be fallible and a reliable memory process might sometimes produce incorrect but coincidentally true beliefs, which prompts skepticism about whether this is knowledge.**, **A belief formed through a reliable memory process but without corroborative evidence or awareness might not intuitively seem to be knowledge.**
