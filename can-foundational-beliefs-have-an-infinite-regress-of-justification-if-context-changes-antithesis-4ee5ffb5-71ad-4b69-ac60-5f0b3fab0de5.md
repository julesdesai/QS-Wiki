# Can foundational beliefs have an infinite regress of justification if context changes?

**Node Type:** antithesis
**Node ID:** 4ee5ffb5-71ad-4b69-ac60-5f0b3fab0de5
**Depth:** 4
**Terminal:** No

**Parent:** [Foundationalism with Reliable Basic Beliefs](foundationalism-with-reliable-basic-beliefs-synthesis-62bc80a9-ef75-4d49-9578-a29890370bd2.md)

## Content

**A foundational belief is one that does not derive justification from other beliefs.**, **Suppose a foundational belief is established in one context but challenged in another, requiring further support.**, **Intuitively, if a foundational belief can be challenged requiring further support, it suggests a potential regress contradicting foundationalism’s core claim.**
