# What happens if context changes after belief is formed?

**Node Type:** antithesis
**Node ID:** d92ae5d0-d450-42c9-9d3e-9e497bac08be
**Depth:** 4
**Terminal:** No

**Parent:** [Relevant Alternatives Theory](relevant-alternatives-theory-synthesis-d7f52b7c-07b8-44a2-8804-2022a32ed27e.md)

## Content

**Consider a scenario where a person forms a belief with certain alternatives ruled out, but then context changes making new alternatives relevant. In such a scenario, intuitively we would judge that the person no longer knows the proposition, yet nothing intrinsic to their belief has changed.**
