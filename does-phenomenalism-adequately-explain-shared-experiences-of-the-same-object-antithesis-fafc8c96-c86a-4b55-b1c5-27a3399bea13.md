# Does phenomenalism adequately explain shared experiences of the same object?

**Node Type:** antithesis
**Node ID:** fafc8c96-c86a-4b55-b1c5-27a3399bea13
**Depth:** 4
**Terminal:** No

**Parent:** [Phenomenalism](phenomenalism-synthesis-a405dc69-be81-4e08-bb71-e181ff4d4e8a.md)

## Content

**Suppose we have a case in which multiple people perceive the same object, like a table, from different angles and distances, yet they all agree on its nature. Intuitively, we either have to say that the table is a collection of personal sensory experiences or that it has a shared, objective existence. If we say it is just a collection of personal sensory experiences, then we must explain how these experiences can align so consistently among different observers. If we say it has a shared, objective existence, this contradicts the view that objects are just collections of sensory experiences.**
