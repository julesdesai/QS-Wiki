# Is sensitivity necessary for knowledge when a proposition is necessarily true?

**Node Type:** antithesis
**Node ID:** 90a141db-a960-400d-a8d4-7339730fe952
**Depth:** 4
**Terminal:** No

**Parent:** [Sensitivity Theory](sensitivity-theory-synthesis-c636dae2-8d6f-478b-827f-65895cd50362.md)

## Content

**If a mathematical proposition is necessarily true, it cannot be false in any possible world.**, **If a person believes a necessarily true proposition, the condition that if it were false, they would not believe it cannot be tested.**, **Therefore, it is not possible for the sensitivity condition to apply to beliefs about necessarily true propositions.**
