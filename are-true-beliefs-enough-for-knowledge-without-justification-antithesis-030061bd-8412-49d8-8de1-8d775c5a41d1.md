# Are true beliefs enough for knowledge without justification?

**Node Type:** antithesis
**Node ID:** 030061bd-8412-49d8-8de1-8d775c5a41d1
**Depth:** 4
**Terminal:** No

**Parent:** [Finite Infinitism](finite-infinitism-synthesis-ec56b28f-9687-4f6e-b141-48dc2e075636.md)

## Content

**Suppose there is a true belief that lacks justification, like guessing the outcome of a coin flip correctly. Intuitively, this does not qualify as knowledge.**, **A person knows a proposition if the proposition is true implies that mere truth suffices for knowledge, which is problematic when justification is absent.**, **There is a plausible principle that for a belief to be knowledge, it must be both true and justified, not merely true.**
