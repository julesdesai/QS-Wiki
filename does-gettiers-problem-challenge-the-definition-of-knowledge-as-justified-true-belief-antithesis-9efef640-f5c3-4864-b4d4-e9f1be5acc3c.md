# Does Gettier's problem challenge the definition of knowledge as justified true belief?

**Node Type:** antithesis
**Node ID:** 9efef640-f5c3-4864-b4d4-e9f1be5acc3c
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology Summary](virtue-epistemology-summary-synthesis-ebb76c32-727a-4eba-8f42-21c2ee3c5622.md)

## Content

**Consider a scenario where a person sees a broken clock showing the correct time purely by chance. Intuitively, this person does not have knowledge of the time despite having a justified true belief.**, **A justified true belief can still result from lucky coincidence, which intuitively does not constitute knowledge.**
