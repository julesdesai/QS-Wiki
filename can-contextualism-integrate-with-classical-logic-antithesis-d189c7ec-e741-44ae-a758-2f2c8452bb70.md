# Can contextualism integrate with classical logic?

**Node Type:** antithesis
**Node ID:** d189c7ec-e741-44ae-a758-2f2c8452bb70
**Depth:** 4
**Terminal:** No

**Parent:** [Knowledge as Contextualism](knowledge-as-contextualism-synthesis-b56b56df-6d59-4c6a-8de4-4cbefdc8c784.md)

## Content

**Classical logic holds that a proposition cannot be both true and not true at the same time. Contextualism, by allowing knowledge claims to be both accepted and rejected based on context, seems to defy this principle of classical logic.**
