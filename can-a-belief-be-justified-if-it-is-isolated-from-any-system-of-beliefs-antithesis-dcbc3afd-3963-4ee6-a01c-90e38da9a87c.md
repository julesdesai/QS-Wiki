# Can a belief be justified if it is isolated from any system of beliefs?

**Node Type:** antithesis
**Node ID:** dcbc3afd-3963-4ee6-a01c-90e38da9a87c
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism View](coherentism-view-synthesis-4dcf0483-85cc-4ed6-8a5e-86aaec2b7626.md)

## Content

**Consider a belief that is true and seems plausible, but it is isolated from any other belief. Intuitively, we would judge that this belief is not justified even though it is true.**, **There is a plausible principle that justification requires some evidential connection or relation. This principle together with the truth of the isolated belief entails that the belief is unjustified.**
