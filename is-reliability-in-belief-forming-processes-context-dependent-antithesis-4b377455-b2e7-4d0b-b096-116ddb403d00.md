# Is reliability in belief-forming processes context-dependent?

**Node Type:** antithesis
**Node ID:** 4b377455-b2e7-4d0b-b096-116ddb403d00
**Depth:** 4
**Terminal:** No

**Parent:** [Reliabilist Externalism Summary](reliabilist-externalism-summary-synthesis-3207924c-3d5f-4b8a-8088-8161a66f3954.md)

## Content

**Suppose we have a case in which a person uses a coin flip to make decisions in a context where they consistently get correct results by sheer luck; intuitively, reliability in this context does not mean this method is a trustworthy process in reality.**, **The context of the reliability affects whether it provides true justification or merely coincidental correctness.**
