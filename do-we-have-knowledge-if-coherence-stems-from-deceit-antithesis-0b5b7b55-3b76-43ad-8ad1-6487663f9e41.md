# Do we have knowledge if coherence stems from deceit?

**Node Type:** antithesis
**Node ID:** 0b5b7b55-3b76-43ad-8ad1-6487663f9e41
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentist Reliability Summary](coherentist-reliability-summary-synthesis-49034568-193c-4a88-8ae7-b062de898be5.md)

## Content

**Consider a scenario where a person’s set of beliefs, including a true proposition, is coherent but was deliberately arranged by someone else using deceit. Intuitively, this undermines the claim that the true belief is genuinely known.**
