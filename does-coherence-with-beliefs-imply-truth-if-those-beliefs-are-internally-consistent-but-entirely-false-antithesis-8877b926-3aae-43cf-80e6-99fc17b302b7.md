# Does coherence with beliefs imply truth if those beliefs are internally consistent but entirely false?

**Node ID:** 8877b926-3aae-43cf-80e6-99fc17b302b7

## Content

**There is a plausible principle that truth requires more than internal coherence; it requires factual accuracy with the external world.**, **An internally coherent set of beliefs could be entirely composed of falsehoods, yet coherence alone cannot equate such a set with knowledge.**
