# Is it possible to have basic beliefs that are not justified?

**Node Type:** antithesis
**Node ID:** c5d76078-1043-4e08-9bc4-2e5e383af426
**Depth:** 4
**Terminal:** No

**Parent:** [Foundationalism with Pragmatic Grounds](foundationalism-with-pragmatic-grounds-synthesis-35d5e861-b788-4e52-a02b-4488aba3b8d2.md)

## Content

**Consider a scenario in which a person's basic belief is formed from arbitrary or unreliable sources. In such a scenario, intuitively, we would judge that the belief is not truly justified.**, **There is a plausible principle that a belief needs to be formed via reliable methods to be justified. This principle together with the idea that basic beliefs require no further justification raises doubt about the reliability of some basic beliefs.**
