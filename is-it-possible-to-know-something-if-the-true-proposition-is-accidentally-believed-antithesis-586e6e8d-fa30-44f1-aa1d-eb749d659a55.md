# Is it possible to know something if the true proposition is accidentally believed?

**Node Type:** antithesis
**Node ID:** 586e6e8d-fa30-44f1-aa1d-eb749d659a55
**Depth:** 4
**Terminal:** No

**Parent:** [Moderate Externalism Summary](moderate-externalism-summary-synthesis-b4668bae-162e-4191-89bc-89b702818630.md)

## Content

**Suppose we have a case in which a person believes a true proposition only because of an accidental factor. Intuitively, we either have to say that the person knows the proposition or merely believes it luckily. If we say the person knows, it contradicts the intuition that knowledge should not be accidental. If we say they merely believe luckily, it challenges the sufficiency of the given conditions for knowledge.**
