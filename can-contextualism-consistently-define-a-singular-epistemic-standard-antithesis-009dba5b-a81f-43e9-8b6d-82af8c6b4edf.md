# Can contextualism consistently define a singular epistemic standard?

**Node Type:** antithesis
**Node ID:** 009dba5b-a81f-43e9-8b6d-82af8c6b4edf
**Depth:** 4
**Terminal:** No

**Parent:** [Contextualism with Variable Justification](contextualism-with-variable-justification-synthesis-53c5a1dc-1391-4651-950b-f21901e771b6.md)

## Content

**Suppose we try to establish one epistemic standard that accounts for all contextual variations. Intuitively, it is difficult to define a singular standard that applies universally yet allows for contextual differences in justification.**, **It is implausible that any one standard could be legitimately flexible enough to encompass all variable contexts without losing coherence.**
