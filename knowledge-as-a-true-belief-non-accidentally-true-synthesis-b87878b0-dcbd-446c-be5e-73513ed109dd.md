# Knowledge as a True Belief Non-Accidentally True

## Content

**Knowledge is a true belief that is non-accidentally true**, **For a person to know a proposition, it is necessary that the proposition is true**, **For a person to know a proposition, it is necessary that they believe the proposition**, **A person knows a proposition if their belief is non-accidentally true given the circumstances**
