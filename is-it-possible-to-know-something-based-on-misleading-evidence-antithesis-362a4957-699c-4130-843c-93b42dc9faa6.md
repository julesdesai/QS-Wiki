# Is it possible to know something based on misleading evidence?

**Node Type:** antithesis
**Node ID:** 362a4957-699c-4130-843c-93b42dc9faa6
**Depth:** 2
**Terminal:** No

**Parent:** [Knowledge as No False Lemmas](knowledge-as-no-false-lemmas-thesis-d0c49c5d-b910-4f5c-9f50-f3147c05c44a.md)

## Content

**Consider a scenario where a person reads in several reliable newspapers that a certain event occurred, which is true, but the newspapers base this claim on a false report that unintentionally got the event right.**, **In such a scenario, intuitively we would judge that the person does not have knowledge, as the evidence was indirectly derived from a falsehood.**
