# Is memory-based knowledge consistent with naive realism?

**Node Type:** antithesis
**Node ID:** ce35b600-ec7e-494d-856b-6cbbae0c3828
**Depth:** 4
**Terminal:** No

**Parent:** [Naive Realism](naive-realism-synthesis-a575a970-7874-461b-b9b5-9de679546681.md)

## Content

**Suppose a person knows a fact through memory of a past perception. Intuitively, they know the fact without currently perceiving the object.**, **If knowledge can be derived from memory without current perception, then directly perceiving objects is not necessary for knowing propositions.**
