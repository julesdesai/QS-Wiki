# How does one distinguish true beliefs from knowledge?

## Content

**Consider a scenario in which a true belief is formed through unreliable means, such as guessing. Intuitively, we would judge that this does not constitute knowledge.**, **A person knows a proposition only if their belief arises from intellectual virtue, distinguishing knowledge from mere true belief.**
