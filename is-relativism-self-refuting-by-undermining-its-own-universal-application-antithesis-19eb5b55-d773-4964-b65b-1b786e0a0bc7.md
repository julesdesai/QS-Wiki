# Is relativism self-refuting by undermining its own universal application?

**Node Type:** antithesis
**Node ID:** 19eb5b55-d773-4964-b65b-1b786e0a0bc7
**Depth:** 4
**Terminal:** No

**Parent:** [Relativism Summary](relativism-summary-synthesis-2665e075-79d4-4adb-9371-55cb5a5794f8.md)

## Content

**Suppose the claim that all knowledge is relative is true within one framework but not another. Intuitively, this raises a contradiction if relativism purports to be universally true.**, **There is a logical fact that claims about universal relativity must themselves avoid absolute assertions.**
