# Does proper functionality address the issue of epistemic luck?

**Node Type:** antithesis
**Node ID:** 3a082aca-79ed-45e1-a657-6f52f9f2eb72
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Functionalism](proper-functionalism-synthesis-aeaab3c5-c7cb-4be4-bb9f-cb5fcc744f10.md)

## Content

**Consider a scenario where a person forms a true belief by coincidence, such as guessing correctly. Intuitively, we judge this as lacking knowledge.**, **Suppose knowledge requires more than just cognitive faculties functioning properly; it requires the exclusion of luck. If proper function allows for beliefs formed via lucky guesses to count as knowledge, the theory struggles.**, **A principle that justification arises from cognitive faculties implies predictability. If outcomes are influenced by luck, unpredictability undermines justification.**
