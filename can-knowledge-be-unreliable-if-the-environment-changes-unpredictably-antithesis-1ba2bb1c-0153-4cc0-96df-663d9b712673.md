# Can knowledge be unreliable if the environment changes unpredictably?

**Node Type:** antithesis
**Node ID:** 1ba2bb1c-0153-4cc0-96df-663d9b712673
**Depth:** 4
**Terminal:** No

**Parent:** [Process Reliabilism with Environment-Sensitive Reliability](process-reliabilism-with-environment-sensitive-reliability-synthesis-cdd770c1-1646-4221-a17f-d7d94aac715e.md)

## Content

**Consider a scenario in which a person knows a proposition using a process reliable in sunny weather, but the weather is prone to abrupt changes without warning.**, **In such a scenario, intuitively, we would judge that knowledge can be compromised if, on occasion, the weather impairs the reliability of the process.**, **This suggests that knowledge should not solely depend on the environment's stability due to unforeseen changes affecting reliability.**
