# Are there cases where false beliefs are deemed safe?

**Node Type:** antithesis
**Node ID:** af7b94b3-5886-4e17-a451-6929f8839752
**Depth:** 4
**Terminal:** No

**Parent:** [Safety Condition Theory](safety-condition-theory-synthesis-b5a72759-42a3-4d9b-b74c-df40c84bdc9d.md)

## Content

**Suppose a person has a false belief that aligns with most nearby possible worlds. Intuitively, this belief is considered safe from error in those worlds.**, **There is a plausible principle that safe beliefs should reliably track the truth, but false beliefs cannot count as knowledge, illustrating a conflict with the safety condition requirement.**
