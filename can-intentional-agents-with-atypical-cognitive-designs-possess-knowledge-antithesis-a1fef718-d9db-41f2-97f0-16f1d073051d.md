# Can intentional agents with atypical cognitive designs possess knowledge?

**Node Type:** antithesis
**Node ID:** a1fef718-d9db-41f2-97f0-16f1d073051d
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Functionalism](proper-functionalism-synthesis-aeaab3c5-c7cb-4be4-bb9f-cb5fcc744f10.md)

## Content

**Consider a case of an artificially designed robot with atypical cognitive faculties producing true beliefs accurately. Intuitively, we might recognize this as knowledge.**, **Justification for knowledge in atypical designs implies that deviations from human-standard cognitive faculties can still yield true beliefs.**, **A principle of knowledge derived from proper function is challenged if atypical faculties, not designed for truth, still reliably produce true beliefs.**
