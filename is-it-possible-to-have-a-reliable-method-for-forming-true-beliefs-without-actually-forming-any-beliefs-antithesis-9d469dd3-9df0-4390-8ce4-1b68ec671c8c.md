# Is it possible to have a reliable method for forming true beliefs without actually forming any beliefs?

**Node Type:** antithesis
**Node ID:** 9d469dd3-9df0-4390-8ce4-1b68ec671c8c
**Depth:** 4
**Terminal:** No

**Parent:** [No-Belief Necessary View](no-belief-necessary-view-synthesis-81f9ccf2-9484-47e2-8ae4-3b9845798b1d.md)

## Content

**A reliable method typically generates beliefs that are reliably true, but if no belief is formed, it's questionable whether the method can be said to be in use.**, **If a method results only in knowledge without belief, it becomes difficult to explain how one could act on or use that knowledge in decision-making.**, **Suppose a person reliably uses a method that results only in knowledge without beliefs; intuitively, one would struggle to distinguish this from mere accidental truth.**
