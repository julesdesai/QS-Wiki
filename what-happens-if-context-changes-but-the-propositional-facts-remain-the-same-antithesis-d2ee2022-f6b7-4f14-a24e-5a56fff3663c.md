# What happens if context changes but the propositional facts remain the same?

**Node Type:** antithesis
**Node ID:** d2ee2022-f6b7-4f14-a24e-5a56fff3663c
**Depth:** 4
**Terminal:** No

**Parent:** [Phenomenal Contextualism](phenomenal-contextualism-synthesis-37be371b-8eaa-49b9-96b8-60f2ca6919b6.md)

## Content

**Consider a scenario in which a person changes their cultural or perceptual context but retains the same factual knowledge about an object, such as the boiling point of water. In such a scenario, intuitively we would be inclined to say the person's knowledge remains valid regardless of context changes.**, **It is implausible that knowledge should be so dependent on context that a change in context results in a different understanding of the same factual information.**
