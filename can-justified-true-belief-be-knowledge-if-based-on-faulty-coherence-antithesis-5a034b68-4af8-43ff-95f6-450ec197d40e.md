# Can justified true belief be knowledge if based on faulty coherence?

**Node Type:** antithesis
**Node ID:** 5a034b68-4af8-43ff-95f6-450ec197d40e
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism with Pragmatic Justification](coherentism-with-pragmatic-justification-synthesis-aef723d7-2be0-4b42-a946-2acb73c17357.md)

## Content

**A situation can occur where a person has a coherent set of beliefs that support each other yet one or more of those beliefs is based on false information.**, **Justification involving a coherent set of beliefs may not ensure that the beliefs are grounded in reality if some beliefs are false.**
