# Can we justify beliefs about future events using evidence?

**Node Type:** antithesis
**Node ID:** 29a00baf-182e-4bde-a156-a55ea68cad8e
**Depth:** 4
**Terminal:** No

**Parent:** [Evidentialist View](evidentialist-view-synthesis-6f63629b-c949-4605-9acc-d91a2708886a.md)

## Content

**Consider the scenario where a meteorologist predicts the weather based on current data. Intuitively, the scientist has justified belief about future weather conditions, yet these beliefs may prove false despite being supported by evidence.**, **There is a plausible principle that predictive beliefs can be justified even if they are not guaranteed to be true, which challenges the evidentialist requirement that justification must make beliefs more likely true than false. This principle suggests justification can exist in uncertain conditions about future events.**
