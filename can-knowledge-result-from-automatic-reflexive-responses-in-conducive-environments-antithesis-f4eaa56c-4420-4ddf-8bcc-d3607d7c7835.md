# Can knowledge result from automatic reflexive responses in conducive environments?

**Node Type:** antithesis
**Node ID:** f4eaa56c-4420-4ddf-8bcc-d3607d7c7835
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-66a6c079-b861-44cf-ac2d-ed1fae76b934.md)

## Content

**Suppose a person's belief is the result of an automatic reflexive response in a truth-conducive environment, such as instinctively dodging an object and forming a belief about its trajectory. Intuitively, we might doubt this counts as knowledge.**, **It is implausible that automatic reflexes, which lack deliberation and reflection, result in knowledge even in conducive environments.**
