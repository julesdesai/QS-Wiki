# Could coherentist phenomenalism fall prey to the "island problem"?

**Node Type:** antithesis
**Node ID:** 2387ca2d-834d-427e-84e0-861a17fd72c6
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentist Phenomenalism](coherentist-phenomenalism-synthesis-7f881170-8b02-4aad-92a8-2bed5a4b8a3c.md)

## Content

**The island problem suggests that justifying a system of beliefs solely from within that system can lead to insular belief clusters.**, **Consider an "island" system that coherently justifies itself using only internal beliefs, without regard to external facts.**, **Lacking a mechanism to relate belief systems to external realities, coherentist phenomenalism could allow isolated and possibly erroneous sets of beliefs to appear justified.**
