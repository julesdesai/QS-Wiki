# Does reliabilism handle the problem of forgotten evidence?

**Node Type:** antithesis
**Node ID:** 9e575257-e9dc-4cdb-806a-895a54eff035
**Depth:** 4
**Terminal:** No

**Parent:** [Reliabilism and the Role of Justification](reliabilism-and-the-role-of-justification-synthesis-e3c23741-1c3a-47b4-8f99-17f5f12eafda.md)

## Content

**Suppose we have a case in which a person recalls an event accurately but forgets the original evidence or basis for the memory yet retains belief based on the event's reliability. Intuitively, we either have to say this belief remains justified or not. If we say it is justified, it implies unwarranted reliance on a forgotten process. If we say it is not justified, it indicates that reliability alone isn't enough without accompanying evidence.**
