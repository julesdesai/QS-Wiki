# How can direct perception account for perceptual relativity?

**Node Type:** antithesis
**Node ID:** 25620be8-c7c0-4995-9ec6-a259456d1317
**Depth:** 4
**Terminal:** No

**Parent:** [Naïve Realism Summary](naïve-realism-summary-synthesis-ad05d8f4-c9d2-4024-8001-bf520360cf02.md)

## Content

**Perceptual relativity refers to how the same object can appear differently to different observers or under different conditions, such as a table appearing rectangular from one angle and trapezoidal from another.**, **If perception is direct, the same object should not produce contradictory perceptual experiences.**, **Perceptual relativity suggests that perception involves interpretative processes, not merely direct awareness.**
