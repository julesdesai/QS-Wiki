# Is a belief-producing process under proper function always truth-conducive?

**Node Type:** antithesis
**Node ID:** 640c988a-705b-4076-ab08-ea8cdc61f13e
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Functionalism](proper-functionalism-synthesis-aeaab3c5-c7cb-4be4-bb9f-cb5fcc744f10.md)

## Content

**Consider a scenario where cognitive faculties are functioning properly according to the design plan but occasionally produce false beliefs due to random error. Intuitively, we would say that not all beliefs formed are knowledge.**, **Suppose a reliable process is characterized by consistent truth production. If occasional errors occur with proper function, then reliability isn't assured.**, **A principle suggesting that cognitive faculties properly functioning reliably produce true beliefs is compromised if proper function allows consistent error.**
