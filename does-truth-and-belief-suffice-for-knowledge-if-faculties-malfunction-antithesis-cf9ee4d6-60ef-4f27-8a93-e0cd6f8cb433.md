# Does truth and belief suffice for knowledge if faculties malfunction?

**Node Type:** antithesis
**Node ID:** cf9ee4d6-60ef-4f27-8a93-e0cd6f8cb433
**Depth:** 4
**Terminal:** No

**Parent:** [Proper Function Theory](proper-function-theory-synthesis-45579dd6-32de-43f3-ae7a-76b3bb7de09b.md)

## Content

**Suppose someone's cognitive faculties malfunction, leading them coincidentally to a true belief. Intuitively, we would not say this constitutes knowledge.**, **A plausible principle is that properly functioning cognitive faculties in a suitable environment are required for knowledge. This principle conflicts with the claim that belief and truth alone are enough.**
