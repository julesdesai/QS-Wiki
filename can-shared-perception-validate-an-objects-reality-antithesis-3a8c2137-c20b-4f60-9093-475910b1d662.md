# Can shared perception validate an object's reality?

**Node Type:** antithesis
**Node ID:** 3a8c2137-c20b-4f60-9093-475910b1d662
**Depth:** 4
**Terminal:** No

**Parent:** [Inter-subjective Phenomenalism](inter-subjective-phenomenalism-synthesis-c00af758-5073-4931-9585-2f560dddd897.md)

## Content

**Suppose there is a scientific hallucination experiment in which subjects see a non-existent object due to manipulation. Intuitively, we either have to say their shared experience confirms the object's reality, which seems mistaken, or deny the reality, which contradicts the idea that external objects can be known via shared perceptual experiences.**, **It is implausible that shared perceptions alone can confirm the reality of external objects without independent verification.**
