# Are external objects entirely determined by perceptions?

**Node Type:** antithesis
**Node ID:** 9cc61838-e0b2-4be6-b551-564a02474e8d
**Depth:** 4
**Terminal:** No

**Parent:** [Pragmatic Phenomenalism](pragmatic-phenomenalism-synthesis-0e894029-9d7b-462b-b98e-b505a0d14728.md)

## Content

**It is implausible that external objects, like physical objects, can be fully reduced to perceptions.**, **Consider a scenario in which multiple people perceive the same object differently, yet the object remains unchanged. Intuitively, the object's properties are not solely defined by perceptions.**
