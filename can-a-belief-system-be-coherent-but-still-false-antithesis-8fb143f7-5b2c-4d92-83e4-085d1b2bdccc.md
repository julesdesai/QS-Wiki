# Can a belief system be coherent but still false?

**Node Type:** antithesis
**Node ID:** 8fb143f7-5b2c-4d92-83e4-085d1b2bdccc
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism](coherentism-synthesis-56ae39cf-eba6-4a99-aaa4-0dbe8a5aac85.md)

## Content

**It is possible for a belief system to be internally consistent yet entirely detached from reality.**, **A set of beliefs may be coherent but still entirely false if it does not correspond to the external world.**, **Coherence alone does not guarantee truth, as a consistent set of beliefs might be based on false premises.**
