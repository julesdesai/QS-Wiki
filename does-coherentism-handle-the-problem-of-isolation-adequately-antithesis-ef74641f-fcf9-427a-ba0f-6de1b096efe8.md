# Does coherentism handle the problem of isolation adequately?

**Node Type:** antithesis
**Node ID:** ef74641f-fcf9-427a-ba0f-6de1b096efe8
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism Summary](coherentism-summary-synthesis-14364920-f24e-48ed-b988-c7d3265113e2.md)

## Content

**Suppose we have a case in which a person believes a set of coherent but isolated beliefs with little to no connection to reality, such as a consistent story about fictional characters. Intuitively, coherence within these isolated beliefs appears insufficient for justification because no external link to the actual world is present.**, **It is implausible that mere internal coherence by itself ensures a belief matches reality, which is crucial for knowledge.**
