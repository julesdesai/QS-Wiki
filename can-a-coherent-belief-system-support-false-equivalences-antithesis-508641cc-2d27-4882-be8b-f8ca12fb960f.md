# Can a coherent belief system support false equivalences?

**Node Type:** antithesis
**Node ID:** 508641cc-2d27-4882-be8b-f8ca12fb960f
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism focuses on systematized justification](coherentism-focuses-on-systematized-justification-synthesis-825f84fd-be44-4c37-9d4c-4b4f5d772a40.md)

## Content

**Suppose we have a case where a coherent set of beliefs produces equivalence between widely divergent concepts like luck and knowledge. Intuitively, such a case implies a failure to distinguish between knowledge and mere belief.**, **It is implausible that coherence itself can act as the sole arbiter in differentiating knowledge quality without reference to evidential truth.**
