# Is knowledge really just true belief formed by intellectual virtue?

**Node Type:** antithesis
**Node ID:** c5f6b2a6-8c40-4c98-bef7-17b0e973980a
**Depth:** 4
**Terminal:** No

**Parent:** [Virtue Epistemology Summary](virtue-epistemology-summary-synthesis-460b628b-7c65-45e3-9b14-3e9060c40e11.md)

## Content

**There is a plausible principle that knowledge requires more than true belief and an intellectual virtue, such as a reliable process, because true belief can arise from luck.**, **Consider a scenario in which a person believes it will rain tomorrow because a psychic told them, and it happens to rain. Intuitively, we would not say the person has knowledge.**, **The intellectual character of the agent seems insufficient if the belief-forming process is unreliable, even if it accidentally yields truth.**
