# Does the Safety Condition Theory allow for true belief based on luck?

**Node Type:** antithesis
**Node ID:** 8e17959b-5a37-4ab3-933d-b0a9fc811130
**Depth:** 4
**Terminal:** No

**Parent:** [Safety Condition Theory](safety-condition-theory-synthesis-89579919-7815-4da3-898a-02510d3efc48.md)

## Content

**Consider a scenario in which a person forms a belief that it will rain tomorrow based solely on a random guess without any reliable evidence, and it turns out to rain. Intuitively, we would judge that the person's belief is true but based on luck rather than knowledge.**, **There is a plausible principle that knowledge cannot be based purely on luck. Based on this principle, a belief that happens to be true due to luck should not count as knowledge.**
