# Does an isolated belief system know by its own coherence?

**Node Type:** antithesis
**Node ID:** ad49a21c-98af-48e5-8d57-66765308d422
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentism Revised](coherentism-revised-synthesis-de283b0a-5fd8-4f4c-bf89-d22a34799ca8.md)

## Content

**Consider a scenario in which a person living in a simulated environment develops a coherent set of beliefs about the world within the simulation. Intuitively, we would judge that these beliefs do not give the person knowledge about the actual world.**, **It is implausible that coherence within an isolated set of beliefs alone suffices for knowledge because it may lack correspondence with an external reality.**
