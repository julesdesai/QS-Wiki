# Can a true belief be knowledge if it does not cohere with a reliable belief system?

**Node Type:** antithesis
**Node ID:** cccc11c5-fe19-4345-aa57-06831d4e557a
**Depth:** 4
**Terminal:** No

**Parent:** [Coherentist Reliability Summary](coherentist-reliability-summary-synthesis-49034568-193c-4a88-8ae7-b062de898be5.md)

## Content

**Consider a scenario in which a person has a true belief based on a lucky guess, but it does not fit within a coherent set of other beliefs. Intuitively, we would judge that this person does not have knowledge.**, **It is implausible to claim that a belief is knowledge if it is coherent with a system of beliefs that includes inconsistencies or falsehoods.**
